# Web用户界面 - 规范

## ADDED Requirements

### Requirement: 前端技术栈

系统应采用现代化的Web界面技术栈。

#### Scenario: 技术栈选择

- **WHEN** 开始前端开发
- **THEN** 使用 React + TypeScript + Tailwind CSS
- **AND** 使用 Vite 作为构建工具
- **AND** 使用 React Router 进行路由管理
- **AND** 使用 Axios 与后端API通信

### Requirement: 页面结构

系统应包含以下核心页面。

#### Scenario: 主导航

- **WHEN** 用户访问系统
- **THEN** 显示顶部导航栏，包含：
  - Logo和系统名称
  - 导航菜单：首页、目标股池、持仓管理、交易记录、实时监控、策略配置
  - 右侧：当前时间、信号提醒图标

### Requirement: 首页/仪表盘

系统应在首页展示账户概览和关键指标。

#### Scenario: 首页展示

- **WHEN** 用户访问首页
- **THEN** 显示以下内容：
  - 账户总览卡片：可用资金、持仓市值、总资产、今日盈亏
  - 持仓分布饼图
  - 近期信号列表（最近5条）
  - 快捷操作按钮：执行扫描、添加持仓

### Requirement: 目标股池页面

系统应展示和管理每日选股结果。

#### Scenario: 目标股列表

- **WHEN** 用户访问目标股池页面
- **THEN** 显示表格，包含：
  - 股票代码、股票名称
  - 策略类型、信号理由
  - 计划买入价、计划仓位
  - 状态（待买入/已买入/已卖出/已放弃）
  - 操作列：设置计划价、标记买入、标记放弃
- **AND** 支持按状态筛选
- **AND** 支持设置计划买入价和仓位弹窗
- **AND** 支持批量操作

#### Scenario: 目标股详情

- **WHEN** 用户点击目标股行
- **THEN** 展开显示详细信息：
  - K线走势图（近30日）
  - 策略信号详情
  - 历史买卖记录（如果有）

### Requirement: 持仓管理页面

系统应展示当前持仓和盈亏情况。

#### Scenario: 持仓列表

- **WHEN** 用户访问持仓管理页面
- **THEN** 显示表格，包含：
  - 股票代码、股票名称
  - 持仓数量、成本价
  - 当前价、浮动盈亏、盈亏率
  - 持仓天数、持仓类型
  - 操作列：添加交易记录、查看详情

#### Scenario: 持仓详情弹窗

- **WHEN** 用户点击持仓行
- **THEN** 显示详情弹窗：
  - 持仓基本信息
  - 盈亏走势图
  - 卖点提醒设置
  - 交易历史

### Requirement: 交易记录页面

系统应展示历史交易流水。

#### Scenario: 交易列表

- **WHEN** 用户访问交易记录页面
- **THEN** 显示表格，包含：
  - 交易日期、股票代码、股票名称
  - 交易类型（买入/卖出）
  - 成交价格、成交数量
  - 手续费、印花税
  - 备注
- **AND** 支持按日期范围筛选
- **AND** 支持按股票代码筛选
- **AND** 支持按交易类型筛选

### Requirement: 实时监控页面

系统应展示盘中实时监控和信号提醒。

#### Scenario: 监控面板

- **WHEN** 用户访问实时监控页面
- **THEN** 显示以下内容：
  - 监控开关：启动/停止实时监控
  - 目标股实时涨跌幅列表
  - 异动股票列表（短线精灵）
  - 信号提醒日志

#### Scenario: 信号提醒

- **WHEN** 监控触发信号条件
- **THEN** 显示桌面通知
- **AND** 在页面右上角显示提醒徽章
- **AND** 播放提示音（可选）

### Requirement: 策略配置页面

系统应支持启用/停用策略和调整参数。

#### Scenario: 策略列表

- **WHEN** 用户访问策略配置页面
- **THEN** 显示策略列表，包含：
  - 策略名称、策略类型
  - 启用/停用开关
  - 参数配置
  - 最近执行结果

#### Scenario: 策略参数配置

- **WHEN** 用户点击策略参数
- **THEN** 显示参数配置弹窗
- **AND** 支持保存参数

## UI/UX 规范

### 配色方案

| 用途 | 颜色 | Hex |
|------|------|-----|
| 主色 | 深蓝色 | #1E3A5F |
| 次色 | 浅蓝色 | #3B82F6 |
| 强调色 | 橙红色 | #F97316 |
| 背景色 | 深灰 | #0F172A |
| 卡片背景 | 灰蓝 | #1E293B |
| 文本色 | 白色 | #F8FAFC |
| 涨色 | 红色 | #22C55E |
| 跌色 | 绿色 | #EF4444 |

### 字体

- 主字体：Inter, system-ui, sans-serif
- 数字字体：JetBrains Mono, monospace

### 组件规范

- 按钮：圆角8px，主按钮使用强调色
- 卡片：圆角12px，带轻微阴影
- 表格：斑马纹，深色表头
- 弹窗：居中，圆角16px，带遮罩

### 响应式

- 桌面端：> 1024px
- 平板端：768px - 1024px
- 移动端：< 768px（基础支持）
