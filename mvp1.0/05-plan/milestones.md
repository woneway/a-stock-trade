# 里程碑计划

## MVP 阶段划分

| 版本 | 功能范围 | 预估时间 |
|------|----------|----------|
| MVP | 收盘后选股 + 目标股池 + 交易计划 + Web界面 | 2周 |
| V1.0 | 盘中实时监控 + 涨跌幅提醒 + 卖点监控 | 1个月 |
| V1.1 | 短线精灵 + 持仓管理增强 + 胜率统计 | 1.5个月 |

## MVP 周计划

| 周次 | 任务 |
|------|------|
| 第1周 | 项目初始化、后端架构、数据库设计、数据获取 |
| 第2周 | 策略实现、API开发、前端页面、集成测试 |

## 风险与预案

| 风险 | 影响 | 预案 |
|------|------|------|
| akshare数据源不稳定 | 信号生成失败 | 多数据源备份，本地缓存 |
| 策略过拟合 | 实盘效果差 | 保守参数，实盘验证 |
| API限速 | 数据获取慢 | 增加间隔，优化请求 |
