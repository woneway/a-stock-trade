# UI/UX 设计规范

## 设计工作流

```
Excalidraw 草图          →    Figma 高保真          →    前端实现
(确认信息架构/交互思路)        (视觉/组件/间距)            (React组件)
```

- **Excalidraw**：快速画出页面区块、信息层级、交互流程，重点确认"放什么、怎么排"
- **Figma**：基于确认的草图做高保真设计，定义精确间距、组件样式、交互细节
- 草图文件存放在 [wireframes/](wireframes/README.md)，每个页面的设计确认点见该目录

## 设计原则

| 原则 | 说明 |
|------|------|
| 深色护眼 | 深色背景适合长时间盯盘 |
| 信息密度高 | 游资需要快速获取信息，布局紧凑 |
| 涨跌色 | 红色=上涨，绿色=下跌（A股习惯） |
| 响应快速 | 数据实时刷新，操作即时反馈 |

## 组件规范

| 组件 | 样式 |
|------|------|
| 按钮 | 圆角8px，主按钮蓝色，次按钮灰色 |
| 卡片 | 圆角12px，背景#1E293B，轻微阴影 |
| 表格 | 深色表头，斑马纹，行hover高亮 |
| 输入框 | 圆角8px，深色背景，边框#334155 |
| 标签 | 圆角16px，小号字体 |
| 弹窗 | 居中，圆角16px，背景遮罩 |
