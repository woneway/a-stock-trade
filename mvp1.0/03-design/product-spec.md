# 产品需求规格（PRD）- v7版本

## 产品概述

- **产品名称**：A股交易系统
- **产品定位**：个人游资超短线交易辅助系统
- **核心场景**：复盘 → 交易计划 → 监控 → 交易记录 → 持仓管理

---

## 页面结构

| 页面 | 路径 | 功能 |
|------|------|------|
| Dashboard | / | 账户概览、今日计划、持仓汇总、信号提醒 |
| 复盘 | /review | 市场复盘 + 我的计划复盘 |
| 交易计划 | /plan | 创建/管理交易计划 |
| 监控 | /monitor | 计划股监控、信号提醒 |
| 交易记录 | /records | 历史交易、胜率统计 |
| 持仓 | /positions | 当前持仓、盈亏、可用数量 |
| 设置 | /settings | 风控、通知、主题 |

## 交易模式定义

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 龙头 | 做市场最高板 | 情绪上升期 |
| 首板 | 0连板做1进2 | 启动期 |
| 二板 | 1连板做2进3 | 确认期 |
| 反包 | 昨日烂板/炸板股反包 | 分歧转一致 |
| 低吸 | 龙头股回调低吸 | 回调企稳 |
| 半路 | 上涨中继买入 | 强势股 |

---

## 风控规则

| 规则 | 默认值 | 说明 |
|------|--------|------|
| 单笔最大仓位 | 20% | 单只股票最大仓位 |
| 日亏损上限 | 5% | 当日亏损达到此值停止交易 |
| 默认止损 | -5% | 默认止损比例 |
| 默认止盈 | +10% | 默认止盈比例 |
| 最大持仓数 | 5只 | 最多持有5只股票 |

---

## 数据流

```
复盘 → 制定计划 → 监控信号 → 买入持仓 → 卖出记录 → 复盘总结
                ↓
           止损/止盈提醒
```
