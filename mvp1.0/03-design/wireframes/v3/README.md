# v3 版本说明

## 改进内容

### 1. 复盘页面增强
- 新增"我的计划"复盘区域 - 复盘自己的交易计划执行情况

### 2. 交易计划页面重构
- **标的类型**：股票/ETF/可转债
- **计划要素**：
  - 标的代码+名称
  - 买入条件（技术面+逻辑）
  - 卖出条件（止盈+止损）
  - 仓位（几成仓）
  - 题材/逻辑
- **状态区分**：
  - 观察中（策略产出，尚未买入）
  - 持仓中（已买入持有）
  - 已完成（止盈/止损卖出）

### 3. 持仓页面增强
- 新增"可用数量"列（考虑T+1限制）

## 页面结构

- 00-dashboard.excalidraw - 仪表盘

- 03-monitor.excalidraw - 监控
- 04-records.excalidraw - 交易记录
- 05-positions.excalidraw - 持仓（增强）
