# v6 版本说明

## 改进内容

### 交易计划模型重构

参考游资做法，完整交易计划应包含：

| 字段 | 说明 | 示例 |
|------|------|------|
| 模式 | 交易模式 | 龙头/首板/二板/反包/低吸/半路 |
| 标的 | 代码+名称+类型 | 600519 贵州茅台 股票 |
| 买入时机 | 什么时机买入 | 十点前/分歧转一致/低开低吸 |
| 验证条件 | 买入前需验证什么 | 成交量>5亿/换手>10% |
| 买入价 | 目标买入价格 | 1700元 |
| 仓位 | 几成仓 | 1成仓 |
| 止损价 | 止损位置 | 1562元 (-5%) |
| 止盈价 | 止盈位置 | 1870元 (+10%) |
| 持仓周期 | 计划持有天数 | 1天/2天/3天 |
| 题材/逻辑 | 买入逻辑 | AI算力龙头/业绩预增 |
| 状态 | 当前状态 | 观察中/持仓中/已完成 |

### 模式分类（游资核心模式）

1. **龙头战法** - 做市场最高板
2. **首板** - 0连板做1进2
3. **二板** - 1连板做2进3
4. **反包** - 昨日烂板/炸板股反包
5. **低吸** - 龙头股低吸回调
6. **半路** - 上涨中继买入
