# v7 版本说明

## 主要改进

### 1. 交易计划模型重构
完整交易计划字段：
| 字段 | 说明 |
|------|------|
| 模式 | 龙头/首板/二板/反包/低吸/半路 |
| 标的 | 代码+名称 |
| 类型 | 股票/ETF/可转债 |
| 买入时机 | 十点前/分歧转一致/低开低吸 |
| 验证条件 | 成交量>5亿，换手率>10% |
| 买入价 | 1700元 |
| 仓位 | 1成仓 |
| 止损价 | 1562元 (-5%) |
| 止盈价 | 1870元 (+10%) |
| 持仓周期 | 1-2天 |
| 题材/逻辑 | AI算力龙头，技术形态突破 |

### 2. 复盘页面增强
- 新增"我的计划复盘"区域
- 显示计划执行情况

### 3. 持仓页面增强
- 新增"可用"列（考虑T+1限制）
- 显示：持仓数量 | 可用数量（T+1）

### 4. 设置页面
- 风控设置：单笔仓位、日亏损、止损止盈、最大持仓数
- 通知设置：信号/成交/止损提醒开关
- 主题设置：深色/浅色切换

### 5. 全局搜索
- 顶部搜索框，支持股票代码/名称搜索

## 页面结构
- 00-dashboard.excalidraw - 仪表盘
- 01-review.excalidraw - 复盘（含我的计划复盘）
- 02-plan.excalidraw - 交易计划（完整模型）
- 03-monitor.excalidraw - 监控
- 04-records.excalidraw - 交易记录
- 05-positions.excalidraw - 持仓（含可用数量）
- 06-settings.excalidraw - 设置
