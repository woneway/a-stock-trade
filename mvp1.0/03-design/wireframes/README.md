# MVP 原型图

个人游资交易系统，MVP版本。

## 版本说明

### v1 - 核心版本
5个核心页面：复盘、交易计划、监控、交易记录、持仓

### v2 - Dashboard版本
- 新增Dashboard页面
- 设置入口

### v3 - 交易计划重构
- 标的类型：股票/ETF/可转债
- 状态区分：观察中/持仓中/已完成

### v4 - 设置页面
- 风控配置
- 保存功能

### v5 - 通知+主题+搜索
- 通知设置
- 主题切换
- 全局搜索

### v6 - 交易模型重构
- 模式：龙头/首板/二板/反包/低吸/半路
- 买入时机、验证条件、持仓周期

### v7 - 完整实现（当前）
- 交易计划完整模型
- 复盘含"我的计划"
- 持仓含可用数量
- 设置页面完整

## 文件结构

```
wireframes/
├── v1/  - 5个核心页面
├── v2/  - +Dashboard
├── v3/  - +交易计划重构
├── v4/  - +设置页面
├── v5/  - +通知/主题/搜索
├── v6/  - +交易模型
└── v7/  - 完整实现
```
