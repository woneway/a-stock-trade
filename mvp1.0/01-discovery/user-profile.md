# 用户画像与痛点分析

## 目标用户画像

| 维度 | 描述 |
|------|------|
| 投资经验 | 3年以上A股交易经验 |
| 交易风格 | 短线/超短线为主，偏好游资打法 |
| 持仓周期 | 1-3天为主，偶尔当日T+0 |
| 止损纪律 | -7%硬止损 |
| 止盈目标 | 5-10%+，涨停不板走 |
| 选股数量 | 每日5-10只目标股 |
| 资金规模 | 50万以下个人账户 |
| 技术能力 | 具备基本电脑操作能力 |

## 核心痛点

| 痛点 | 描述 | 严重程度 |
|------|------|----------|
| 复盘效率低 | 每次需要手动查看大量股票数据 | 高 |
| 策略不成体系 | 交易经验难以系统化、复用 | 高 |
| 监控不及时 | 人工盯盘无法覆盖所有目标股 | 高 |
| 记录混乱 | 手动记录容易遗漏、错误 | 中 |
| 复盘困难 | 难以统计胜率、收益曲线 | 中 |
