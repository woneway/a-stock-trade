# 现有工作流分析

## 当前操作流程

1. 人工复盘选股（耗时30分钟-1小时）
2. 在券商APP手动下单
3. 手动记录交易到Excel/笔记

## 核心业务流程

```
收盘后复盘(15:30) → 制定次日计划 → 盘中信号监控 → 人工执行 → 卖出 → 复盘
```

## 游资每日复盘流程（顶级标准）

```
1. 回顾当日操作 → 逐笔复盘买卖点、仓位管理
2. 换手率选股 → 盯换手率排名前十的个股
3. 板块晋级分析 → 盯"首板→二板"晋级路径
4. 昨日涨停今日表现 → 按成交金额排序，看大资金方向
5. 当日涨停复盘 → 全部涨停股加入自选，分类到板块
6. 炸板股复盘 → 思考下次如何避免
7. 龙虎榜复盘 → 跟踪大资金动向
8. 跌停榜复盘 → 识别风险信号
9. 制定次日策略 → 潜在买入标的、卖出时机、仓位管理
```

## 游资三种交易模式

| 交易模式 | 入场时机 | 风险收益 | 核心技巧 |
|----------|----------|----------|----------|
| **低吸** | 强势股回调至均线企稳 | 低风险中等收益 | 只做强势回调、缩量企稳、分批建仓 |
| **半路** | 股票冲板途中介入 | 中风险中高收益 | 盘感要求高、需板块共振、量价配合 |
| **打板** | 涨停瞬间买入 | 高风险高收益 | 只打首位板/1进2/2进3、带动板块优先 |
