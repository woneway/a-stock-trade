## 二、改进方案

### 2.1 复盘页面改进

**新增数据项**:

```typescript
interface ReviewData {
  // 大盘数据 (新增)
  shIndex: number;      // 上证指数
  szIndex: number;      // 深证指数
  cyIndex: number;      // 创业板指

  // 资金数据 (新增)
  mainFlow: number;     // 主力净流入(亿)
  instBuy: number;      // 机构买入(亿)
  instSell: number;    // 机构卖出(亿)

  // 情绪评分 (新增)
  sentimentScore: number; // 0-100

  // 炸板数据 (已有但需完善)
  breakUpCount: number;
  breakUpLoss: number;
}
```

### 2.2 数据库模型扩展

```python
class HistoricalPlan:
    # 现有字段...

    # 新增字段
    sh_index: float      # 上证指数
    sz_index: float      # 深证指数
    cy_index: float      # 创业板指

    main_flow: float    # 主力净流入(亿)
    inst_buy: float     # 机构买入(亿)
    inst_sell: float     # 机构卖出(亿)

    hot_sectors: JSON    # 热门板块
    limit_up_stocks: JSON # 涨停股票
    break_up_stocks: JSON # 炸板股票
    inst_research: JSON   # 机构调研
```
