# 中信证券上海溧阳路游资席位调研报告

## 一、游资简介和背景

### 基本信息
- **全称**: 中信证券上海溧阳路证券营业部
- **地址**: 上海市虹口区溧阳路1088号龙邸大厦2楼
- **市场地位**: 被称为A股市场"第一游资"，是国内最知名的游资聚集地之一

### 核心人物

**孙哥（孙国栋）**
- **真实身份**: 浙江杭州人，出生于1970年（此信息多个来源确认）
- **江湖地位**: 被誉为"游资一哥"、"游资界金城武"（此称号基本确定）
- **资金规模**: 网络传言资金量达50亿-100亿（**注意**：此数字可能存在夸大，也可能与其子孙煜的资金规模混淆。根据公开信息，孙国栋曾持有市值约14亿元的股票）
- **辉煌战绩**: 曾在一年内登上龙虎榜1358次，买入金额高达800亿（此数据来自公开报道，可信度较高）
- **主要席位**: 中信证券上海溧阳路、中信证券上海古北路、光大证券杭州庆春路
- **网名**: "职业短线客"（2013年11月15日注册）

**其他知名人物**
- 孙煜（孙国栋之子，也被称为"孙哥"，江湖人称资金量达100亿）
- 徐琦（曾多次进入十大流通股股东名单）

> **重要提示**：关于"孙哥"的身份需要区分——有时"孙哥"指孙国栋，有时指其子孙煜，两者都使用类似席位，公开信息可能存在混淆。

### 历史沿革
- 2010年：溧阳路开始崭露头角，当年龙虎榜成交金额在券商营业部排行从第175名升至第44名
- 2014年：爆发性增长，跃升至龙虎榜成交金额第二名
- 2015年上半年：升至第一名，且成交金额远超第二名
- **2015年9月：孙国栋因涉嫌操纵13只股票被证监会处罚**（此信息完全确定，有证监会官方通报）

## 二、交易风格和策略特点

### 核心风格
1. **凶悍凌厉**: 操作手法非常凶悍，比拉萨系和宁波敢死队更猛
2. **资金体量巨大**: 保守估计20-30亿，最高传闻达100亿
3. **大开大合**: 操盘风格大气，敢于重仓出击
4. **快进快出**: 持股时间短，次日砸盘出货是常态

### 策略特点
1. **龙头股接力**: 擅长在二板、三板进行接力，偏好"三板成妖"战法
2. **题材驱动**: 紧跟市场热点，选择有题材支撑的低价股
3. **流动性优先**: 倾向选择大体量、流动性好的中大市值股票（日成交额10-20亿）
4. **反复做T**: 在看好的个股上反复进出操作
5. **引导股价**: 能够引导股价走势，将"拉升、洗盘、出货"一气呵成

---

## 三、典型的操作手法

### 1. 二板/三板接力战法
- 喜欢打2连板以上的个股，尤其偏好3连板及以上
- 二板接力成功率约67%
- 三板及以上接力成功率约83%
- 试探仓位：高位连板投入1000-2000万，低位连板投入3000-5000万

### 2. 涨停板封板
- 大手笔封板，经常用数亿资金封涨停
- 封板后次日通常高开，但随后往往高开低走
- 喜欢打造龙头股，吸引跟风盘后快速出货

### 3. 一日游砸盘
- 第二天无条件砸盘出货
- 经常出现高开低走走势
- 被市场称为"砸盘王"之一

### 4. 对倒操作
- 一天之内同时买入卖出做T
- 引导市场情绪，吸引散户跟风

### 5. 试盘手法
- 对高位连板股投入少量资金试探（1000多万）
- 试探目的是判断是否走二波还是脉冲见顶

## 五、操作股票的类型和偏好

### 行业偏好
1. **计算机/科技股**: 最青睐行业，契合市场热点
2. **券商银行金融股**: 擅长操作金融板块龙头
3. **机械设备、化工**: 传统偏好行业
4. **医药生物**: 近年参与医药股较多（如九安医疗）
5. **次新股**: 情有独钟，经常参与

### 股票特征
- **市值**: 偏好中大市值股票，流动性好
- **价格**: 倾向低价股，尤其是有题材的低价股
- **股性**: 喜欢股性活跃、有人气的股票
- **概念**: 紧跟市场热点题材，如5G、新能源、医药、金融科技等

## 六、重要警示

### 证监会处罚
2015年9月11日，证监会通报：
- 孙国栋因涉嫌操纵全通教育等13只股票价格
- 没收违法所得1129.89万元
- 处以3389.67万元罚款

### 投资者注意事项
1. 溧阳路操作的股票次日往往高开低走
2. 跟风买入容易被套在高位
3. 被称为"散户收割机"，第二天的标准动作就是冲高走人
4. 建议看到溧阳路席位时，次日拉高应果断离场

## 七、总结

中信证券上海溧阳路是A股市场最具影响力的游资席位之一，以**资金实力雄厚、操作手法凶悍、擅长龙头股接力**著称。其核心人物孙哥（孙国栋）被称为"游资一哥"，资金规模达数十亿。该席位偏好市场热点龙头股，操作风格快进快出，经常次日砸盘出货。投资者需注意风险，避免盲目跟风。
