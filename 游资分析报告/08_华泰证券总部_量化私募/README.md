## 华泰证券总部（0503）游资席位调研报告

### 一、游资简介和背景

华泰证券总部是华泰证券股份有限公司的总部营业部，在龙虎榜上通常显示为"华泰证券股份有限公司总部"。

**关键数据：**
- 2021年龙虎榜**百强营业部第三名**
- 剔除机构席位后，近一年**上榜次数2184次**，**成交金额738亿**
- 平均每次上榜成交金额超过3000万元
- 2023年上半年龙虎榜营业部成交额冲到**行业第二**，仅次于中金公司上海分公司

**背景定位：**
根据开源证券金工团队的研究报告，通过数据回撤和股东行为推演，华泰证券总部被锁定为**量化私募的游资席位**。与另外三家（中金财富北京宋庄路、中金国际上海黄浦区湖滨路、招商证券深圳深南东路）并列为最可疑的量化私募席位。

---

### 二、交易风格和策略特点

1. **策略类型**：采用**中高频策略**，每天交易股票数量基本在20家以上

2. **仓位特点**：
   - 交易金额多在**1000万以内**
   - 超过3000万的占比基本不到10%
   - 一旦确认为龙头，会**敢于重仓**

3. **股票偏好**：
   - 偏好**上市3年以上的老股票**
   - 市值：主打**20-30亿之间的小盘股**，其次是50亿以上的大盘股
   - 价格：**中低价股**为主，大部分不超过30元

4. **技术形态偏好**：
   - 擅长短线交易，主打**上升周期的强势股**
   - 倾向于**45度角的稳定上行行情**
   - 偏好**平缓推升后的加速阶段**
   - 大涨之后**高位回调的股票**也是其抄底时机

---

### 三、典型的操作手法

华泰证券总部的操作手法可分为两类：

**1. 套利模式（主要）**
- 每天买入大量个股
- 包含**低吸超跌**和**打首板**两种
- 仓位都不多，基本在1000万左右级别
- 基本以**套利为主**，有一两个点收益就走

**2. 龙头模式**
- 一旦某只个股确认为龙头（连板龙头或趋势龙头）
- 会**敢于重仓加大仓位**
- 特别注意**三板以上龙头**
- 经常在开板当日进场，承接抛压
- 然后**博弈二波**

   - 4连板后次日进场
   - 4月26日2835万进场
   - 4月28日2337万止损

2. **华润双鹤**（新冠药阿兹夫定概念股）

   - 4月27日再次买入4400万，卖出4900万

### 五、操作股票的类型和偏好总结

| 维度 | 偏好特征 |
|------|----------|
| 上市时间 | 3年以上老股票 |
| 市值 | 20-30亿小盘股为主，其次50亿以上大盘股 |
| 价格 | 中低价股，30元以下为主 |
| 形态 | 45度角稳定上行、平缓推升后加速、强势股回踩 |
| 概念 | 市场热点题材、龙头股、趋势股 |
| 策略 | 套利为主（低吸超跌、打首板）+ 龙头重仓 |

---

### 总结

华泰证券总部是一个**以量化交易为主的游资席位**，具有以下显著特点：
- 交易**高频、分散**，每天操作20+只股票
- 单笔金额**相对较小**（1000万以内为主）
- 擅长做**短线套利**
- 也会在确认龙头后**重仓博弈**
- 偏好**中低价、小盘、上市久**的股票

该席位的操作风格相对稳健，属于"不见兔子不撒鹰"类型，平时以小仓位套利为主，但一旦抓住龙头机会会果断重仓。
