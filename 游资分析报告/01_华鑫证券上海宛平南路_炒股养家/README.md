## 二、交易风格与策略特点

### 2.1 核心交易风格

| 特点 | 详细描述 |
|------|----------|
| **通道优势** | 利用华鑫证券VIP通道抢入一字涨停板，被市场誉为"一字板之王" |
| **大局观强** | 对市场情绪周期有精准判断，善于在行情启动初期重仓入场 |
| **正派风格** | 不会直接"核按钮"（恐慌性抛售），有一定席位溢价 |
| **资金体量大** | 单票投入通常占当日成交额的5%-10%，单个股票持仓可达数亿元 |

### 2.2 情绪周期理论

"炒股养家"的核心投资理念是**"基于市场情绪的揣摩"**，其理论体系主要包括：

1. **行情生于冰点，死于一致**：市场恐慌至极时是最佳入场时机，群体亢奋时需果断离场
2. **情绪周期六阶段**：试错→确认→发酵→高潮→分歧→退潮，只参与确认期至高潮期
3. **买于分歧，卖于一致**：在热点题材初期或龙头股首次放量分歧时介入

### 2.3 仓位管理

- **风险收益比优先**：强调"没有100%的把握不满仓"
- **分仓策略**：单只个股持仓分散至2-3个席位，避免过度曝光
- **主账户（宛平南路）**：主打趋势加速与龙头接力
- **次账户（红宝石路）**：侧重次新股与题材潜伏

## 三、典型操作手法

### 3.1 主要战法

1. **一字板抢筹**
   - 利用VIP通道在次新股或重组股未开板前锁定筹码
   - 适用于新股上市、重大资产重组等确定性题材

2. **连板战法**
   - 专门做三连板及以上的妖股接力
   - 偏好换手充分的连板，烂板出妖股

3. **加速板**
   - 针对趋势龙头，高位加仓引导市场合力
   - 如剑桥科技、鸿博股份等

4. **反核低吸**
   - 对恐慌错杀股逆向抄底
   - 在市场恐慌时入场捡便宜筹码

5. **回马枪战法**
   - 首板回调后缩量三阴下跌，不破涨停板低点
   - 下影线逐渐变长，说明有主力悄悄吸筹

### 3.2 操作纪律（养家六条铁律）

1. 大赚后空仓
2. 连亏3天空仓
3. 分歧日不交易
4. 高开5个点以上不追
5. 高位放巨量不追
6. 强势接力，弱势低吸
