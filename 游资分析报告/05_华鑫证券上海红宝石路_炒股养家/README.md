## 一、游资简介和背景

### 1.1 席位归属

华鑫证券上海红宝石路证券营业部是**知名游资"炒股养家"的核心常用席位之一**。该营业部位于上海市长宁区红宝石路500号，隶属于华鑫证券有限责任公司。

### 1.2 炒股养家本人

- **真实姓名**：林广昌（也有资料称其为"张吉惟"，存在不同说法）
- **籍贯**：上海人，70后
- **资金规模**：保守估计数十亿级别（2015年便已突破10亿）
- **出道时间**：90年代末入市，2008年正式辞职职业炒股

## 二、交易风格和策略特点

### 2.1 核心理念：情绪周期驱动

炒股养家的核心逻辑是**"基于市场情绪的揣摩"**，将市场划分为：

- **赚钱效应阶段**：连板股数量多、涨停股次日溢价率高
- **亏钱效应阶段**：核按钮（跌停）频率高、热门股次日闷杀比例大

### 2.2 情绪周期四阶段操作策略

| 阶段 | 仓位比例 | 操作重点 |
|------|----------|----------|
| 启动期 | 20%-30% | 试错龙头 |
| 发酵期 | 50%-70% | 聚焦核心股 |
| 高潮期 | 30%-50% | 去弱留强 |
| 退潮期 | 0%-10% | 空仓观望 |

### 2.3 主要风格特点

1. **通道优势明显**：利用华鑫证券的通道速度抢入一字板个股
2. **独到市场理解**：对市场和个股具有深刻的判断力
3. **大局观强**：敢于在行情启动初期重仓入场
4. **格局较大**：不会直接"核按钮"离场，是市场情绪的维护者
5. **善于基本面投机**：对题材的选择非常精准
6. **注重空间**：股票空间至少30%才会买进

## 三、典型的操作手法

### 3.1 核心战法

#### （1）一字板抢筹
利用通道优势，在次新股或重组股未开板前锁定筹码，使个股一字涨停，隔日高位逐步离场。

#### （2）龙头战法
- **龙头股筛选标准**：
  - 题材强度：优先选择国家政策、技术革命相关标的
  - 市场地位：个股具备首板启动快、涨停封单量大于流通盘5%、无机构重仓等特征
- **龙头操作四步法**：
  1. 选龙：首板/二板锁定潜在龙头
  2. 试仓：启动期用10%-20%仓位试错
  3. 加仓：确认龙头地位后加至50%以上仓位
  4. 离场：板块后排集体跌停或龙头高位缩量加速时撤退

#### （3）买于分歧，卖于一致
- 在热点题材初期或龙头股首次放量分歧时介入
- 当市场情绪高度亢奋、散户跟风盘涌入时果断离场

### 3.2 养家心法精髓

1. **高手买入龙头，超级高手卖出龙头**
2. **敢于在大盘低位满仓，敢于在大盘高位空仓**
3. **别人贪婪我更贪婪，别人恐慌我更恐慌，永远顺势而为**
4. **买股集中，出手少，但一出手就是重仓**
5. **不让赚钱的票变成亏损**

### 5.2 个股选择标准

1. **题材强度**：优先选择国家政策支持、技术革命相关标的
2. **市场地位**：要求个股具备领涨气质

4. **基本面**：善于研究研报，基本面投机
5. **流动性**：偏好有足够流动性的热门股

### 5.3 操作特点

- **重仓出击**：买股集中，一出手就是重仓
- **格局锁仓**：敢于在龙头股上锁仓待涨
- **多手法并用**：低吸、反包、打板、主升浪、半路点火等各种模式
- **协同作战**：常与华鑫证券其他席位联动操作

## 六、总结与评价

华鑫证券上海红宝石路作为"炒股养家"的核心席位，在游资界具有极高的知名度和影响力。其特点可以概括为：

1. **实力雄厚**：资金量极大，是市场顶级游资之一
2. **风格稳健**：格局大，不轻易核按钮，有一定席位溢价
3. **善于引导**：对市场情绪有独到理解，能够打造和锁仓龙头
4. **通道优势**：利用华鑫证券的通道速度抢入一字板

对于普通投资者而言，跟踪该席位的龙虎榜操作可以作为市场情绪参考，但需注意其操作风格较为激进，且席位溢价并非绝对保证投资收益。
