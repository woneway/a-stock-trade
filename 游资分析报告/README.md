## 游资分类总结

### 一、情绪周期派
- **代表人物**：炒股养家（林广昌）
- **核心席位**：华鑫证券上海宛平南路、上海红宝石路
- **核心理念**：基于市场情绪的揣摩，把握情绪周期各阶段
- **主要战法**：一字板抢筹、龙头战法、买于分歧卖于一致
- **适合投资者**：有较强市场感知能力和纪律性的短线投资者

### 二、散户集中营
- **代表人物**：拉萨天团（东方财富散户群体）
- **核心席位**：拉萨团结路、拉萨东环路
- **核心理念**：高抛低吸、追涨杀跌、超短线交易
- **主要战法**：T+0操作、大阴线买入、一日游
- **特点**：本质是散户聚集，而非单一游资

### 三、稳健派
- **代表人物**：章建平（章盟主）
- **核心席位**：国泰君安上海江苏路
- **核心理念**：中短线结合，趋势投资与短线结合
- **主要战法**：龙头战法、二波定江山、反复做T
- **特点**：出货温和，被称"善庄"

### 四、凶悍派
- **代表人物**：孙国栋（孙哥）
- **核心席位**：中信证券上海溧阳路
- **核心理念**：龙头股接力，快进快出
- **主要战法**：二板/三板接力、一日游砸盘
- **特点**：操作凶悍，次日经常高开低走

### 五、综合派
- **代表人物**：国信泰然九路团队
- **核心席位**：国信证券深圳泰然九路
- **核心理念**：周密计划，间歇性操作
- **主要战法**：新股/次新股、重组题材
- **特点**：11个团队各自为战，完善风控

### 六、量化交易派
- **代表人物**：量化私募
- **核心席位**：华泰证券总部
- **核心理念**：中高频策略，套利为主
- **主要战法**：低吸超跌、打首板、龙头重仓
- **特点**：交易分散，单笔金额小

### 七、龙头战法派
- **代表人物**：赵强（赵老哥）
- **核心席位**：中国银河证券绍兴
- **核心理念**：只看"人气"，追逐热点题材
- **主要战法**：打板、二板定龙头、过顶擒龙
- **特点**：8年1万倍，擅长龙头股操作

### 八、低吸派
- **代表人物**：乔帮主
- **核心席位**：招商证券深圳蛇口工业七路
- **核心理念**：强势股回调低吸，复利思维
- **主要战法**：均线支撑低吸、反包涨停、尾盘操作
- **特点**：以低吸为主，配合打板，风格稳健

## 投资建议

1. **跟踪游资席位可以作为市场情绪参考**，但不应作为唯一投资依据
2. **不同游资风格差异大**，需根据自己的风险承受能力选择学习对象
3. **游资操作高风险高收益**，普通投资者盲目模仿风险较大
4. **席位溢价并非绝对**，需结合市场环境和个股基本面综合判断
5. **建议学习游资的交易思路和纪律性**，而非简单模仿操作手法
