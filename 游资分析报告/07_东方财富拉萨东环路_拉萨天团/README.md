# 东方财富拉萨东环路证券营业部游资研究

## 一、游资简介和背景（与拉萨天团的关系）

### 1.1 拉萨天团的构成

"拉萨天团"是指东方财富证券在拉萨的几个营业部形成的特殊投资者群体，主要包括以下五个席位：
- 东方财富证券拉萨团结路第一证券营业部
- 东方财富证券拉萨团结路第二证券营业部
- 东方财富证券拉萨东环路第一证券营业部
- 东方财富证券拉萨东环路第二证券营业部
- 东方财富证券拉萨金融城南环路证券营业部

### 1.2 形成背景

拉萨天团的形成源于东方财富证券的商业模式特点：

1. **历史渊源**：2015年，东方财富收购西藏同信证券，获得国内证券从业资格。西藏同信证券在拉萨团结路、东环路各有两个营业部。

2. **互联网开户集中分配**：东方财富通过互联网模式开展大量开户业务，这些通过网上开户的账户会自动划分到上述几个营业部，使得这些营业部成为散户的聚集之处。

3. **散户大本营**：由于东方财富网上开户的便捷性，吸引了众多散户投资者。这些散户在开户时并没有经过特殊筛选，只是因为开户流程而被划分到了拉萨的这几个营业部。

因此，**拉萨天团本质上并不是某个游资的专用席位，而是散户大本营**，被市场称为"家人们"。当然，由于任何人包括知名游资都可以在东方财富开户，所以里面也不乏有大小游资暗盘操作。

## 二、交易风格和策略特点

### 2.1 核心特点

| 特点 | 详细说明 |
|------|----------|
| **操作周期短** | 快进快出，博取差价，一日游是常态，最多不超过4天 |
| **追涨杀跌** | 在市场情绪高涨时敢于追涨，推动股价上涨；当市场出现变化时，也可能迅速杀跌 |
| **高抛低吸** | 擅长做T，通过日内的高抛低吸来降低成本、获取利润 |
| **风格激进** | 偏好追涨杀跌，操作手法灵活，热衷于参与市场热点题材的炒作 |
| **频繁交易** | 在龙虎榜中出现频率极高 |

### 2.2 具体操作策略

1. **大阴线买入**：擅于在大阴线买进股票，或者在股价处于低位时吸筹
2. **水下低吸**：利用震荡市场找寻拉升机会
3. **持股策略**：被套会认赔割肉，绝不恋战
4. **利润最大化**：如果能低吸到大长腿，当日有可能获得15%的利润

## 三、典型的操作手法

### 3.1 两种主要风格

**第一种：埋葬游资**
- 跟大游资拼命：当游资拉升时，拉萨就卖出；当游资卖出时，拉萨就买入
- 每次都能把游资玩得团团转
- 高抛低吸玩得贼溜
- 游资喜欢互砸，但拉萨非常团结，经常同时出现3-4个席位

**第二种：顶板连板**
- 当拉萨上板的票涨停后，第二天连板概率很大
- 经常团结顶一字或者直接高开秒板
- 告诉游资：想进来可以涨停板去买，要不就别进来
- 当价位达到一定高度时，放游资进来接盘，拉萨全部撤退

### 3.2 买卖特征

- **买入特征**：经常出现在龙虎榜买入席位，而且是多个席位同时出现
- **卖出特征**：第二天大概率会出货，导致股价下跌
- **对倒操作**：经常出现买卖双方都有拉萨席位的情况，进行对倒操作

## 四、历史上的著名交易案例（龙虎榜）

### 4.1 近期案例

**案例1：ST立方（2026年2月5日）**
- ST立方因日涨幅达到15%登上龙虎榜
- 买卖双方席位高度重合，均被拉萨天团主导
- 买入：东环路第一证券营业部买入421.62万元（买一）
- 卖出：东环路第二证券营业部卖出485.60万元（卖一）

**案例2：风语筑（2026年2月24日）**
- 该股6天4板，股价报12.85元
- 拉萨团结路第二证券营业部买入2336.38万元
- 拉萨东环路第二证券营业部买入2104.95万元

**案例3：光线传媒**
- 拉萨东环路第一证券营业部买入约6343.67万元
- 拉萨团结路第一证券营业部买入约6296.42万元
- 拉萨金融城南环路证券营业部买入约5627.63万元

**案例4：通达电气**
- "拉萨天团"包揽买入榜前5席
- 知名游资炒股养家卖出2201.22万元

## 五、操作股票的类型和偏好

### 5.1 股票类型偏好

| 类型 | 说明 |
|------|------|
| **热门题材股** | 热衷于参与市场热点题材的炒作 |
| **妖股** | 敢于介入走势强劲的妖股 |
| **ST股票** | 经常参与ST股票的炒作 |
| **小盘股** | 偏好中小盘股票，便于操作 |

### 5.2 市场影响

**助涨作用**：
- 当"拉萨天团"集体买入某只股票时，可能会推动股价短期内大幅上涨甚至涨停连板
- 吸引更多投资者关注

**助跌作用**：
- 被市场称为"散户的绞肉机"
- 拉萨天团玩过的股票，经常出现"一地鸡毛、寸草不生"的情况，一路阴跌
- 当四个拉萨席位同时上阵时，第二天大概率吃跌停

---

## 总结

拉萨东环路第一证券营业部和第二证券营业部是"拉萨天团"的核心成员，本质上是东方财富证券的散户聚集地。由于其独特的操作风格（快进快出、追涨杀跌、频繁交易），在市场上形成了独特的"拉萨天团"现象。投资者在遇到拉萨天团时需要格外谨慎，因为他们的操作往往导致股价短期剧烈波动，且以出货为主。

**关键要点**：
1. 拉萨天团不是游资，而是散户大本营
2. 操作周期极短（最多4天）
3. 经常"埋游资"，与游资反向操作
4. 参与后股票往往一路阴跌
5. 四个席位同时出现时，次日大概率下跌
