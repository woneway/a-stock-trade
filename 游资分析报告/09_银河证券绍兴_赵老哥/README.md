# 中国银河证券绍兴游资席位深度调研报告

## 一、游资简介与背景

### 1.1 席位基本信息

中国银河证券股份有限公司绍兴证券营业部位于浙江省绍兴市。该营业部因承载知名游资"赵老哥"的主要交易席位而闻名于A股市场，是中国股市最具影响力的游资聚集地之一。

### 1.2 赵老哥其人（重要说明：以下信息来自网络传闻和公开报道）

**赵老哥身份确认**：
- **真实姓名**：赵强（2017年通过浙江震天律师事务所发表声明，确认自己是"赵老哥"本人）
- **出生年份**：网络传言为1987年，浙江绍兴人
- **毕业院校**：网络传言毕业于浙江财经大学（具体信息无法完全核实）
- **外形特征**：身材瘦小，戴着眼镜，温文尔雅

**成长历程（来自网络传闻）**：
- 2007年：携带约10万元本金入市
- 2015年4月16日：在淘股吧发帖《八年一万倍》，提到"资金终于上了一个大台阶，感谢中国神车！"
- 2015年：凭借中国中车战役，资金达到10亿级别

> **重要提示**：赵强本人从未公开确认其具体财富数字、初始本金、操作记录等详细信息。网络上流传的"从10万到10亿"等说法多为市场推测和传言，**无法独立核实**。赵强本人曾声明，从未注册过以"赵老哥"命名的微信公众号、微博账号或QQ号，市场上存在大量冒用其名义的骗子账号。

### 1.3 常用席位一览

赵老哥团队的常用席位分布广泛，形成了多席位协同操作的格局。根据市场公开信息，其主要席位包括：中国银河证券绍兴营业部（核心主力席位）、中国银河证券浙江分公司、中国银河证券北京阜成路证券营业部、浙商证券绍兴解放北路营业部、银泰证券上海嘉善路营业部、华泰证券股份有限公司浙江分公司等。多席位协同操作的优势在于能够分散交易风险、规避监管关注，同时在不同市场环境下灵活切换主攻席位。

## 二、交易风格与策略特点

### 2.1 核心交易理念

赵老哥的交易哲学建立在对市场情绪的深刻理解之上，他曾明确表示："短期交易，不讲价值，不讲技术，只讲故事。如果一定要捏造技术的话，就是看'人气'，一口气上不来就要注意跑。有故事，有大量资金活跃的票，就有肉吃，吃完撤退，慢的买单。人生难得几回搏，热点来了全力以赴，热点退潮果断卖出。热点就是印钞机。"这段话精辟地阐述了其交易体系的核心逻辑——紧跟市场情绪、追逐热点题材、在龙头股中寻找超额收益。

赵老哥还特别强调题材更新的重要性，他主张"有新题材，坚决抛弃旧题材。只有新题材，才能凝聚市场里最敏锐、最犀利的那股资金。这股资金，以亿为单位。所到之处，只有涨停。只要能跟上这股资金，想不赚钱都难"。这种对题材轮动的敏锐把握，使其能够在不同的市场周期中持续捕获涨停板机会。

### 2.2 龙头战法体系

赵老哥最为人所知的交易方法论是"龙头战法"，其核心可以概括为以下几个方面：第一，"二板定龙头"，即通过观察第二个涨停板来初步判定龙头候选股，因为第一个涨停板往往难以准确判断股票的真正潜力，只有经历换手检验的二板才具备成为龙头的资质；第二，第一时间发现市场的精神领袖（龙头），第一时间买入并持有，直到发现新的精神领袖后果断切换；第三，龙头必须爆量，接受群众的检验，分歧爆量换手回封才能渡劫成功；第四，操作上要先预测、后跟随。

在具体选股标准上，赵老哥偏好以下特征的股票：首板必须是有换手的实体板，一字板或T字板被剔除，低位未出现大幅拉升的较好；二板也必须是有换手的实体板，对于一字涨停开盘的直接剔除；二板有打开过的较好，但开板回落幅度不大（1%至3%最佳），最后再封住形成放量的实体板。

### 2.3 资金管理铁律

赵老哥在资金管理方面展现出严格的纪律性，其核心原则包括：分仓原则方面，单日开仓不超过3只股票（除绝对龙头外），以分散单一持仓风险；动态止盈方面，断板当日减半仓，破5日线清仓，确保利润落袋为安；黑名单制度方面，淘汰股性差、有减持计划的个股，避免潜在的地雷风险。在仓位调节上，赵老哥会根据市场周期阶段进行动态调整：启动期仓位50%、发酵期仓位80%、高潮期仓位30%、退潮期空仓休息。

### 2.4 不同资金规模的策略差异

值得注意的是，赵老哥的交易策略会根据资金规模进行相应调整。在市值达到1亿元之前，他倾向于选择长期盘整后突然放量拉升、突破盘整平台的小盘股，采取尾盘买入策略，第二天如果高开就密切观察分钟指标，一旦跌破警戒线、上涨无力就迅速止盈走人，一般持股时间仅1至3天，如果低开则认赔止损走人。当市值超过1亿元以后，他的操作策略转变为：要么逢低买入中国中车等有题材的蓝筹大盘股，股价拉升后走人；要么每天打涨停板，第二天如果高开就先卖出一半利润落袋，何时卖出另一半看时机情况，第二天低开同样认赔止损走人。

## 三、典型操作手法

### 3.1 打板战术

打板是赵老哥最核心的操作手法，即在股票涨停的瞬间或涨停板上买入。其打板战术的核心在于准确判断股票的上涨趋势，并在关键时刻迅速行动。在实施打板战术时，赵老哥通常遵循以下步骤：首先，根据自选股票池和热点板块筛选出具有上涨潜力的个股；其次，通过技术分析、量价关系等手段分析股票的买卖点；再次，在确认买入信号后迅速下单买入；最后，在持有股票期间设置止损位，控制风险。这种打法要求交易者具备极强的盘面分析能力、快速的决策能力以及严格的纪律性。

### 3.2 过顶擒龙

"过顶擒龙"是赵老哥另一种常用的短线波段操作方法。其操作逻辑是：股价从60天均线下方冲过60天线，见到第一个高点后慢慢回落到60天线附近，再次启动时，当股价涨到前高附近要引起高度关注。当股价以放量长阳过前头（最好是涨停）时，一轮快速拉升行情就可能启动。"过顶擒龙"的含义包括：股价突破前期高点，处于无套牢盘阶段，容易产生更多新高和巨大赚钱效应，吸引跟风盘追买；股价此时量价都处于高控盘状态，主力多头资金对股价影响巨大，容易产生加速上涨大行情。

### 3.3 龙头—补涨—切换三位一体战法

赵老哥在资金规模过亿后独创了"龙头—补涨—切换"三位一体战法，即在持有龙头股的同时，布局同板块的补涨股，一旦发现新的题材龙头出现，立即切换持仓。这种战法的好处在于既能享受龙头股带来的超额收益，又能通过补涨股分散风险，同时保持对市场热点的持续跟踪。

### 3.4 反包与龙回头

赵老哥还擅长运用反包和龙回头战术。在股票短期回调后，他会在关键技术支撑位买入，等待股价再次上涨创出新高。这种战术要求交易者对个股的股性有深入了解，能够准确判断回调的幅度和结束时间。

## 四、历史上的著名交易案例

### 4.1 中国中车战役（2015年）——封神之战

中国中车战役是赵老哥职业生涯最具标志性的战役，也是其"八年一万倍"传奇的顶点。2015年，中国南北车合并重组为"中国中车"，被视为"中国神车"，成为当年大牛市的风向标之一。2015年4月，赵老哥在中国中车的操作中展现出惊人的胆识和判断力。

在这场战役中，各路顶级游资纷纷参与，包括章盟主、孙哥、职业炒手王涛、徐晓、马信奇以及总舵主徐翔等。然而，赵老哥的表现最为亮眼：4月9日第二波中途一字板开板时，赵老哥加仓1.7亿元；4月15日继续加仓3.3亿元；4月20日见顶大跳水的关键时刻，赵老哥敏锐地察觉到了危险，在近乎最高位果断清仓出局。凭借这一战，赵老哥两周内狂赚5亿元，资金体量突破20亿元大关，完成了从2亿元到10亿元的跨越。

2015年4月16日，赵老哥在淘股吧发表了那篇著名的帖子《八年一万倍》，正文仅有27个字："今天是值得纪念的一天，资金终于上了一个大台阶，感谢中国神车！！！"这篇帖子成为了中国股市游资史上永恒的经典。

### 4.2 东方通信战役（2018年-2019年）

2018年年底，华为5G概念横空出世，东方通信成为蹭上5G概念的跨年妖股，股价从4元左右一路上涨至41.58元。赵老哥此时动用银河绍兴、阜成路以及银泰嘉善路等席位，12月28日买入东方通信3041万元。很快，东方通信便开启了6天5涨停的强势上涨之路，赵老哥再次展现了其对龙头股的精准把握能力。

### 4.3 九安医疗战役（2021年-2022年）

九安医疗是新冠检测概念的超级大牛股，股价在短期内暴涨数倍。在这场战役中，赵老哥多次参与操作。根据龙虎榜数据，赵老哥曾买入九安医疗4900万元，参与了该股的炒作。此外，赵老哥还多次在九安医疗上做T（日内差价），展现出灵活的操作风格。

### 4.4 其他经典战役

赵老哥在其他多只知名牛股上都有亮眼表现，包括：2013年外高桥自贸区行情，3个月获利3倍；春兴精工操作中与多路游资协同，形成良好的板块效应。

## 五、操作股票的类型与偏好

### 5.1 股票类型偏好

赵老哥在股票选择上具有明确的偏好特征。首先，他偏好具有强烈故事性和想象空间的题材股，无论是业绩反转、并购重组还是概念炒作，只要有"故事"可讲，就有可能进入其选股视野。其次，他青睐市场热度高、资金活跃的股票，认为"有故事，有大量资金活跃的票，就有肉吃"。再次，他注重股票的股性活跃度，喜欢历史上有过涨停基因的股票，这类股票往往在短期内更容易走出强势行情。

### 5.2 行业与题材偏好

从历史操作记录来看，赵老哥对以下几类题材有明显偏好：第一，政策利好驱动的题材，如2015年的一带一路（中国中车）、2018年的5G（东方通信）；第二，业绩反转或爆发性增长的公司，如九安医疗；第三，新兴产业和热点概念，如新能源汽车、光伏等；第四，大盘蓝筹股（在资金规模壮大后），尤其是在有明确催化剂的情况下。

### 5.3 流通市值偏好

在资金规模较小时，赵老哥偏好小盘股，因为小盘股更容易被资金推动涨停，也便于快速进出。随着资金规模的扩大，他逐渐参与中大盘股的操作，如中国中车这种大盘蓝筹。这种转变反映了他在不同资金规模下的策略调整能力。

## 六、风险控制与纪律性

尽管赵老哥以激进的打板风格著称，但他在风险控制方面展现出极强的纪律性。他严格设置止损位，一旦股票走势不及预期，会果断割肉离场，避免深度套牢。他曾表示"顺势而为，再好的股票，再好的预期，若没有按自己的预判上涨，要么不参与，若参与了就要抛弃一切幻想，果断止损止盈"。这种严格的纪律性是其能够实现稳定复利的重要因素。

## 总结

中国银河证券绍兴营业部作为赵老哥的核心席位，承载了中国股市一段传奇般的游资神话。赵老哥从10万元本金起步，在8年间实现了1万倍的收益增长，达到10亿元级别，其成就足以比肩世界顶级投资大师。他的成功绝非偶然，而是建立在其独特的交易哲学、严格的纪律性以及对市场情绪的深刻理解之上。其"龙头战法"、"二板定龙头"等方法论影响了一代短线交易者，成为中国股市超短交易领域的标杆性人物。然而，需要特别指出的是，游资的高收益模式建立在高风险操作之上，普通投资者盲目模仿往往面临巨大风险，赵老哥的成功经历难以简单复制。
