# 赵老哥完整交易系统

## 一、赵老哥龙头战法的详细内容

### 1.1 核心思想

赵老哥龙头战法的核心理念：**"只做龙头，只做主升，只做惯性"**

- **二板定龙头，一板能看出来个毛**：首板涨停的股票很多，但真正能成为龙头的需要在第二板继续强势涨停才能初步确认。
- **有大龙头就做龙回头，没有大龙头就做反弹**
- **龙头股特点：涨得早、涨得猛、涨得久、回调少**

### 1.2 龙头股的五个基本条件

1. **必须从涨停板开始**：不能涨停的股票不可能做龙头
2. **低价优先**：龙头股一般不超过10元
3. **流通市值适中**：适合大资金运作，一般选择50亿-100亿市值的股票
4. **有板块效应**：能够带动整个板块上涨
5. **股性活跃**：历史上常有涨停记录，人气足，号召力强

### 1.3 龙头股的识别特征

- **最先启动**：在板块中第一个涨停，涨幅最大
- **换手充分**：不是一字板，而是经过充分换手的涨停板
- **逆市抗跌**：在大盘下跌末端或市场恐慌时逆市涨停
- **放量攻击**：具备攻击性放量特征
- **筹码供不应求**：走势呈现越走越轻、加速上涨的特征

### 1.4 龙头股的六个买点

1. **突破板**：突破重要压力位的首板
2. **创新高板**：加速上涨的首板
3. **反弹板**：触底反弹或回踩均线反弹的首板
4. **二板**：二板定龙头的确认板
5. **反包板**：3天2板的反包涨停
6. **龙回头**：龙头股回调后的第二波

## 二、二板定龙头的具体选股方法

### 2.1 核心理念

"二板定龙头"是赵老哥成名绝技。一个板块启动时，会诞生很多首板涨停股，而龙头的诞生就是从众多首板里面经过市场考验淘汰后选出来的。

### 2.2 二板定龙头的五步策略

1. **筛选20天内有二连板的个股**：但要注意不能是一字板（需要充分换手）
2. **观察回调**：随后股价必须出现至少三天连续回调下行的阴线走势，回调时间不能太长，尽量在一周内才有效
3. **回调幅度**：回调的最大位置不能跌破第一个涨停板的一半
4. **寻找买点**：当股价回调阶段开始出现阳线，并且成交量加倍放大，出现倍量柱，配合市场热点风口
5. **持有标准**：只要股价不破10日均线就是安全的

### 2.3 二板选股的具体标准

- **龙头标签**：越前排的涨停股越有持续性，尽量接力龙头前三，不追龙头跟风股
- **超级主力资金**：超级单+紫旗+主力动向紫柱
- **市值低**：连板股以游资接力为主，市值尽量低于100亿，50亿以下更佳
- **属于主线风口板块**：支线连板概率低，空间小

### 2.4 二板选股公式（通达信参考）

```
XG:COUNT(C/REF(C,1)>1.097 AND C=H,5)>=3 
AND EVERY(V<REF(V,1),3) 
AND CROSS(MA(C,5),MA(C,10)) 
AND C>REF(C,1)*0.95;
```

## 三、赵老哥的打板技巧和纪律

### 3.1 打板的核心原则

打板手法强调**"快、准、狠"**

#### 打板最看重的要素：

1. **股票趋势**：注重分析股票的长期趋势
2. **买卖点判断**：精准把握股票的买卖点
3. **资金流动**：关注资金流向
4. **风险控制**：设置止损点

### 3.2 打板的具体标准

1. **封板时间**：上午十点半前封板的股票最好
2. **低价优先**：低价股成为龙头的可能性大得多
3. **分时走势**：封板时直线快速拉升，或者凌厉的三段式拉升
4. **股性要好**：龙头应该是股性很活的票

### 3.3 赵老哥八大铁律

1. 任何一只个股，只要高位跌了9天，一定要及时跟进
2. 任何一只个股，只要连续涨了2天，一定要先减仓
3. 任何一只个股，只要拉升超过7%，第二天冲高要卖出
4. 任何一只个股，一定要等到回调结束之后再切回来
5. 任何一只个股，只要三天毫无波澜，再观察三天，不行就换
6. 任何一只个股，如果次日赚不到前一天的成本价，就要出
7. 涨停板，有三必有五，有五必有七，五板成妖
8. 两连板的个股要逢低进，第五天会是一个非常好的卖点

### 3.4 十大操作纪律

1. 开盘不接力：等技术面分时走稳、换手够了再说
2. 只做龙头，只做主升，只做惯性
3. 不看、不买、不研究跟风股
4. 交易一定要纯粹，模式越简单越好
5. 先预测，后跟随
6. 确定性远比利润的多少重要
7. 非热点题材不看、不研究
8. 学会空仓，等待属于自己的机会
9. 热点来了全力以赴，热点退潮果断卖出
10. 牛股要学会做波段敢拿敢做T

## 四、赵老哥过顶擒龙战法

### 4.1 战法原理

> 股价从60天均线下方冲过60天线，见到了第一个高点；然后股价慢慢回落到60天线附近，然后再次启动，当股价涨到前高附近时，要引起我们的高度关注。要求过前头部一定是要放量长阳过前头，最好是涨停，一轮快速拉升行情就开始了。

### 4.2 过顶擒龙的含义

1. **无套牢盘阶段**：股价突破了前期的高点，处于无套牢盘的阶段
2. **高控盘状态**：股价此时量价都处于高控盘状态，主力多头资金对股价的影响巨大
3. **后量超前量**：成交量与价格都有类似的走势，后价超前价，后量超前量

## 五、赵老哥的资金管理方法

### 5.1 仓位管理原则

- **早期**：五五模式，两只满仓（5成仓位的股票1，5成仓位的股票2）
- **中期**：五三二模式（5成仓位主流板块，3成仓位次主流，2成仓位备用）
- **后期**：由于资金体量问题，估计很难满仓

### 5.2 持股原则

- 坚持持股不超过3只的模式
- 从来不补仓，买入之后的结果只有两个：止盈和止损
- 不在同一个股票上做T（会做T但核心仓位不做）

### 5.3 分仓策略

将资金分为三个账户：
1. **短炒账户**：用于超短线操作
2. **中长线账户**：用于价值投资
3. **波段账户**：用于中期波段操作

### 5.4 风险控制

- **止损原则**：短线交易不恋战，一旦走势偏离预期，果断止损（通常设置-5%至-7%的止损线）
- **空仓等待**：没有明确机会时保持空仓
- **分散持仓但不过度分散**：主攻1-2只核心标的

### 5.5 止盈方法

1. **静态止盈**：设立具体的盈利目标位，一旦到达盈利目标位时坚决止盈
2. **动态止盈**：
   - 价格回落幅度：股价与最高价相比减少5%-10%时止盈卖出
   - 均线破位止盈：股价掉头击穿均线时止盈
   - 技术形态止盈：出现滞涨并构筑头部形态时坚决止盈

## 六、赵老哥对龙回头的运用

### 6.1 龙回头的核心前提

1. **首波是板块真龙头**：至少3连板以上，有板块带动性
2. **回调期间缩量**：不跌破首波启动关键支撑（5日/10日线为主）
3. **回调时间**：3-10个交易日，太久容易凉

### 6.2 龙回头的选股标准

1. 前期有2-3个涨停板（代表前期是一个龙头股）
2. 拉升过程中有量能积累
3. 回落调整过程中极度缩量
4. 股价回落至重要支撑位：10日均线或20日均线
5. 调整时间3-10个交易日

### 6.3 龙回头的三种实战公式

**公式一：缩量回调企稳二波**
```
XG:COUNT(C/REF(C,1)>1.097 AND C=H,5)>=3 
AND EVERY(V<REF(V,1),3) 
AND CROSS(MA(C,5),MA(C,10)) 
AND C>REF(C,1)*0.95;
```

**公式二：强势反包二波**
```
XG:COUNT(C/REF(C,1)>1.097,8)>=4 
AND REF(C/REF(C,1)<=0.93,1) 
AND C/REF(C,1)>1.097 
AND V>REF(V,1)*1.8 
AND C=H;
```

**公式三：趋势不破二波**
```
XG:COUNT(C/REF(C,1)>1.097,8)>=4 
AND EVERY(C>MA(C,10),5) 
AND V<REF(V,1) 
AND C>REF(C,1);
```

## 七、赵老哥核心心法总结

### 关于选股
- 二板定龙头，一板能看出来个毛
- 必须新题材龙头
- 跟风股，不看、不买、不研究

### 关于题材
- 有新题材，坚决抛弃旧题材
- 市场差不怕，成交额少也不怕，就怕题材多
- 题材多势必切换快

### 关于操作
- 交易一定要纯粹，模式越简单越好，做加法容易，做减法难
- 先预测，后跟随
- 确定性，远比利润的多少重要

### 关于龙头
- 大龙头都是多点共振的结果：题材派、技术派都认同
- 只做龙头、只做主升、只做惯性
- 没有爆量的都不能说是龙头，既然是龙头必须爆量，接受群众的检验
