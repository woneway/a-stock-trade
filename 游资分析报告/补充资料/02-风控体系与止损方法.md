# 游资风控体系与止损方法

> 本文件整理自网络公开资料，总结游资的风控体系和止损方法。

---

## 一、风控优先级原则

```
时间止损 > 价格止损
流动性止损 > 技术止损
环境止损 > 个股止损
```

---

## 二、止损方法详解

### 1. 时间止损
- **游资对持仓时间的敏感度远高于价格波动**
- 铁律：**隔日开盘30分钟后未达预期强度立即砍仓**
- 超过指定时间没有表现就要走，不能硬扛

### 2. 流动性止损
- 当买一档位堆积量骤减80%时
- 尤其涨停封单金额骤减80%，且题材处于非常一致的高位时
- **不计成本出货**

### 3. 环境止损
- 当市场出现天地板数量或一字跌停闷杀数量≥3家
- 尤其市场总龙头也天地板时
- **需要空仓观望**

### 4. 固定金额/比例止损
- **单笔交易风险敞口≤账户净值2%**
- **日累计亏损限额≤5%**
- **周最大回撤≤10%**
- 止损点一般设在买入成本的**5%-10%**

### 5. 技术分析止损
- 股价跌破重要支撑位时执行止损
- 移动平均线作为动态支撑参考
- 关键K线形态出现时触发止损

---

## 三、顶级游资的止损铁律

1. "跌破关键支撑位立即离场"
2. "在监管红线面前，果断抽身远比硬扛更明智"
3. "宁可少赚，也要控制回撤"
4. "止损要坚决，不能犹豫"
5. "错了就要认，认了就要改"

---

## 四、风控核心原则

| 原则 | 具体要求 |
|------|----------|
| 仓位控制 | 单票不超30%，单日不超70% |
| 回撤控制 | 日亏5%必须停手，周亏10%必须空仓 |
| 流动性 | 只做成交额亿以上的股票 |
| 环境判断 | 大盘不好时空仓或轻仓 |
| 止损执行 | 到了止损位无条件执行 |

---

## 五、风控检查清单

### 买入前检查
- [ ] 仓位是否符合计划
- [ ] 是否有明确止损位
- [ ] 下跌空间是否可控
- [ ] 市场环境是否支持

### 持仓中检查
- [ ] 是否到止损位
- [ ] 环境是否恶化
- [ ] 流动性是否充足
- [ ] 是否需要减仓

### 卖出后检查
- [ ] 是否执行了止损/止盈
- [ ] 仓位是否需要调整
- [ ] 是否需要总结经验
