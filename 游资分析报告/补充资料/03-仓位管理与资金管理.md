# 游资仓位管理与资金管理

> 本文件整理自网络公开资料，总结游资的仓位管理和资金管理方法。

---

## 一、三种经典仓位管理法

### 1. 金字塔形仓位管理法

```
初始建仓：4-5成仓
上涨后加仓：3成仓
继续上涨：2成仓
最后一次：1成仓
```

- **优点**：按报酬率控制仓位，胜率越高仓位越高
- **缺点**：震荡行情中较难获得收益
- **适合**：保守型投资者

### 2. 漏斗形仓位管理法

```
初始建仓：1-2成仓
下跌后加仓：2-3成仓
继续下跌：3-4成仓
最后加仓：满仓
```

- **优点**：初始风险小，不爆仓情况下盈利可观
- **缺点**：方向判断错误时仓位越来越重，可能爆仓
- **适合**：热衷抄底的激进投资者

### 3. 矩形仓位管理法

```
每次建仓：固定比例（如30%）
每次加仓：遵循固定比例
成本平均分摊
```

- **优点**：逐步抬高持仓成本，平均分担风险
- **缺点**：初期成本上升较快，易陷入被动
- **适合**：行情启动同时跟进的投资者

## 二、游资打板仓位管理策略

### 根据市场趋势调整仓位

| 市场环境 | 推荐仓位 |
|---------|---------|
| 牛市上升趋势 | 70%-80% |
| 震荡市 | 30%-50% |
| 熊市下降趋势 | 20%以下或空仓 |

### 根据风险承受能力调整

| 风险承受类型 | 推荐仓位 |
|-------------|---------|
| 低风险承受者 | 30%-40% |
| 高风险承受者 | 50%-60% |

### 分散投资原则

- 不要把所有资金集中在一只股票
- 按不同行业、不同规模公司分配资金
- 单票持仓不超过总资金的30%

## 三、资金管理核心原则

### 1. 风险敞口控制
- **单笔交易风险敞口≤账户净值2%**
- **日累计亏损限额≤5%**
- **周最大回撤≤10%**

### 2. 仓位分配原则

```
强势市场：
  - 龙头股：30-40%
  - 补涨股：20-30%
  - 备用金：30-40%

震荡市场：
  - 龙头股：20%
  - 备用金：80%

弱势市场：
  - 龙头股：10%
  - 备用金：90%
```

### 3. 资金效率原则

- **只做确定性机会**：没有把握时空仓
- **拒绝无效交易**：做的越多，错的越多
- **保留备用金**：永远留有子弹

---

## 四、赵老哥仓位管理

- **早期**：五五模式，两只满仓（5成仓位的股票1，5成仓位的股票2）
- **中期**：五三二模式（5成仓位主流板块，3成仓位次主流，2成仓位备用）
- **后期**：由于资金体量问题，估计很难满仓

---

## 五、养家心法仓位管理

- **赢面60%以下**：观望
- **赢面60%-70%**：小仓出击
- **赢面70%-80%**：中仓出击
- **赢面80%-90%**：大仓出击
- **赢面90%以上**：满仓

---

## 六、仓位管理检查清单

- [ ] 单票仓位是否超30%
- [ ] 总仓位是否与市场环境匹配
- [ ] 是否预留备用金
- [ ] 是否分散投资
- [ ] 日亏损是否超5%
- [ ] 周亏损是否超10%
