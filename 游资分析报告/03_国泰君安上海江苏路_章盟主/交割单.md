## 操作风格总结

| 特点 | 说明 |
|------|------|
| 操作风格大胆 | 经常在涨停板、跌停板等极端行情中逆势操作 |
| 资金体量巨大 | 单日买入卖出金额经常达到数千万元甚至数亿元 |
| 偏好热点题材 | 主要操作热门概念股、AI股、科技股等 |
| 波段操作 | 经常出现在三日连续榜中，体现中短线操作思路 |
| 领涨股操作 | 经常买入涨停板的龙头股，带动市场情绪 |
