## 国泰君安上海江苏路游资席位调研报告

### 一、游资简介和背景

**席位归属**：国泰君安证券上海江苏路证券营业部

**实际控制人**：章建平（绰号"章盟主"、"总舵主"）

**人物背景**（以下信息来自公开报道和网络传闻）：
- **基本信息**：章建平，1967年出生于浙江临安（现杭州市临安区），毕业于天津商学院（现天津商业大学）
- **入市时间**：1996年以5万元资金进入股市（此信息多个来源确认）
- **性格特点**：性格温和、为人低调、生活简朴
- **资金规模**：网络传言目前身家超百亿（有不同估算版本，可参考公开持仓约50亿元）

> **说明**："十大游资之首"更多是江湖称号和口碑体现，并非官方评选结果。章建平的真实财富数字难以精确核实，以上数字仅供参考。

**营业部地位**：
- 国泰君安上海江苏路营业部是国泰君安所有分支机构中规模最大、综合实力最强的旗舰营业部
- 客户资产规模超过1000亿元
- 被称为"亿万富翁制造基地"
- 多年来一直是排名前列的游资上榜席位

---

### 二、交易风格和策略特点

1. **稳健派风格**：与一般游资的快进快出不同，章盟主操作相对沉稳

2. **中短线结合**：根据市场环境灵活切换周期，不拘泥于单一模式
   - 牛市主攻趋势股
   - 震荡市专注短线题材
   - 熊市轻仓或空仓

3. **趋势投资与短线结合**：资金体量上百亿，具有一定的公募基金思维

4. **"善庄"著称**：出货比较温柔，不会猛烈砸盘，跟风资金往往有收益

5. **注重风控**：设置止损位并严格执行，一旦走势不符合预期会及时退出

---

### 三、典型操作手法

1. **龙头战法**：善于在市场热点或新兴题材出现时，迅速锁定并介入龙头股

2. **二波定江山**：极少参与首波拉升，专注龙头股回调后的二次爆发

3. **震荡操作**：在强势上升股上反复做T降低成本，一般翻倍才会止盈出局

4. **重仓出击**：一旦确定目标，会果断重仓买入，单只个股仓位可达30%-50%

5. **与机构合作**：喜欢布局一些中长线的股票，不断买入基本面良好、有成长性的股票

6. **打板进场**：经常在一字涨停板上进场扫货

7. **均线系统**：以30日均线作为趋势判断核心

---

### 四、历史上的著名交易案例（龙虎榜）

1. **中信证券一战成名**
   - 2007年单日动用54个账户操作中信证券，创30亿成交记录

2. **中国铝业大战机构（2007年）**
   - 用10亿元资金在次新股中国铝业上大战机构投资者
   - 章建平做多，公募机构做空
   - 章建平大获全胜，一战封神

3. **北辰实业、招商轮船（2006年）**
   - 2006年10月16日至12月6日，北辰实业大涨172%
   - 章建平所在的湖墅南路营业部累计买入2.5亿元，卖出2亿元
   - 招商轮船上市首日单向买入1.05亿元，随后连续涨停

4. **近年典型案例**：
   - 2023年华大基因：净买入2.72亿元
   - 2024年万科A：净买入1.49亿元
   - 2024年长安汽车：买入1.91亿元
   - 2025年操作万丰奥威：从8元推升至16元
   - 2025年布局寒武纪：持仓市值约35亿元

---

### 五、操作股票的类型和偏好

1. **题材偏好**：
   - 偏重于消息面选股，对热点题材把握准确及时
   - 偏好各类热点题材，对流通市值偏好不明显

2. **市值偏好**：
   - 首选10-20亿的小盘股
   - 其次是50亿以上的大盘股

3. **价格偏好**：
   - 主打低价股，大部分不超10元
   - 中高价次新股偶尔涉及

4. **流动性要求**：
   - 偏好成交额超10亿元的热点龙头股
   - 规避冷门标的

5. **股票类型**：
   - 偏好上市3年以上的老股票
   - 喜欢操作成交额比较大的股票
   - 擅长操作权重大票的主升浪

---

### 总结

国泰君安上海江苏路作为章盟主（章建平）的核心席位，是A股市场最具影响力的游资聚集地之一。章建平从5万元起步做到百亿身家的传奇经历，使其成为游资界的标杆人物。他的操作风格兼具游资的灵活性与机构的严谨性，以"善庄"著称，出货温和，注重为跟风资金留出退出空间。
