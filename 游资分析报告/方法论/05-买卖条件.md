# 买卖条件方法论

## 一、买入条件

### 1. 买入条件清单

#### 情绪条件
- [ ] 当前情绪周期适合买入（启动/发酵/高潮）
- [ ] 赚钱效应良好
- [ ] 无明显亏钱效应

#### 大盘条件
- [ ] 大盘无系统性风险
- [ ] 大盘在20日线之上（强势市场）
- [ ] 涨跌停比例正常

#### 板块条件
- [ ] 有明确的主流板块
- [ ] 板块效应明显（多只涨停）
- [ ] 板块持续性好

#### 个股条件
- [ ] 是板块龙头/最强股
- [ ] 涨停质量好（量价配合）
- [ ] 符合买入模式

---

### 2. 具体买入条件

#### 竞价买入条件
- [ ] 竞价量价匹配
- [ ] 高开符合预期（3-5%）
- [ ] 板块效应强
- [ ] 龙头股竞价强势

#### 低吸买入条件
- [ ] 回调到重要均线（5/10/20日线）
- [ ] 缩量企稳
- [ ] 有支撑
- [ ] 是龙头股

#### 打板买入条件
- [ ] 涨停瞬间买入
- [ ] 板块效应强
- [ ] 是龙头股
- [ ] 封板质量高

---

## 二、卖出条件

### 1. 卖出条件清单

#### 止损条件（必须执行）
- [ ] 亏损达到止损线（一般5-7%）
- [ ] 大盘系统性下跌
- [ ] 板块集体走弱
- [ ] 个股走势不及预期

#### 止盈条件
- [ ] 达到预期涨幅（7-15%）
- [ ] 出现明显顶部信号
- [ ] 板块情绪退潮
- [ ] 龙头股断板

---

### 2. 具体卖出信号

#### 均线卖出法
- [ ] 跌破5日均线（短线）
- [ ] 跌破10日均线（趋势）
- [ ] 跌破20日均线（中期）

#### 量价卖出法
- [ ] 放量滞涨
- [ ] 放量下跌
- [ ] 缩量上涨（量价背离）
- [ ] 高位放量十字星

#### 形态卖出法
- [ ] 出现双头/顶背离
- [ ] 跌破上升趋势线
- [ ] 出现射击之星
- [ ] 连续大阴线

#### 情绪卖出法
- [ ] 板块出现亏钱效应
- [ ] 龙头股断板
- [ ] 跌停板增多
- [ ] 炸板率上升

---

## 三、止盈止损策略

### 1. 固定比例法

| 止盈目标 | 止损线 |
|---------|-------|
| 7% | -5% |
| 10% | -7% |
| 15% | -10% |

### 2. 移动止损法

- 盈利5%后，止损上移到成本价
- 盈利10%后，止损上移到5%盈利
- 盈利15%后，止损上移到10%盈利

### 3. 均线移动止损

- 5日线上持有，破5日线止盈
- 10日线上持有，破10日线止盈
- 20日线上持有，破20日线止盈

---

## 四、不同阶段的卖出策略

### 1. 启动期
- [ ] 达到预期收益就卖
- [ ] 不格局，见好就收
- [ ] 严格止损

### 2. 发酵期
- [ ] 持股待涨
- [ ] 移动止损保护利润
- [ ] 龙头断板再卖

### 3. 高潮期
- [ ] 逐步减仓
- [ ] 锁定利润
- [ ] 不贪心

### 4. 退潮期
- [ ] 第一时间卖出
- [ ] 不犹豫
- [ ] 亏损也要卖

---

## 五、买卖条件检查流程

### 买入前检查
1. 情绪周期适合吗？
2. 大盘安全吗？
3. 有主流板块吗？
4. 是龙头股吗？
5. 符合买入模式吗？

### 买入后检查
1. 走势符合预期吗？
2. 需要设置止损吗？
3. 止盈目标是多少？

### 卖出前检查
1. 达到止盈目标了吗？
2. 出现止损信号了吗？
3. 板块情绪变弱了吗？
4. 大盘有风险吗？

---

## 六、核心纪律

### 买入纪律
1. **不符合条件不开仓**
2. **没有确定性不做**
3. **只做模式内交易**
4. **控制仓位**

### 卖出纪律
1. **止损必须坚决**
2. **止盈可以不卖**
3. **龙头断板必卖**
4. **退潮期不格局**

---

## 七、注意事项

1. **止损是保命符**：亏损5%无条件止损
2. **止盈要果断**：达到目标位可以分批卖
3. **龙头多格局**：龙头股可以多持有
4. **杂毛不格局**：非龙头及时卖
5. **退潮必须跑**：退潮期第一时间离场
