# 量化选股条件方法论

## 一、量化选股核心指标

量化选股是通过设定具体数值条件来筛选股票，主要包括换手率、量比、股价、涨停天数、振幅、均线等技术指标。

---

## 二、各项指标详解

### 1. 换手率

**定义：** 一定时间内股票转手买卖的频率

**选股标准：**

| 场景 | 换手率要求 |
|-----|-----------|
| 正常活跃 | 1-3% |
| 温和上涨 | 3-5% |
| 强势上涨 | 5-10% |
| 涨停板 | 2%左右 |

**实战应用：**
- 换手率 > 3%：股性活跃
- 换手率 > 5%：资金关注度高
- 换手率 > 10%：可能是高位出货
- 换手率 < 1%：股性不活跃

---

### 2. 量比

**定义：** 当日每分钟平均成交量/过去5日每分钟平均成交量

**选股标准：**

| 场景 | 量比要求 |
|-----|---------|
| 正常 | 1-2倍 |
| 上涨健康 | 1.5-3倍 |
| 强势 | 5-10倍 |
| 涨停板 | 2倍左右 |

**实战应用：**
- 量比 > 1：当天成交比过去5天活跃
- 量比 > 2：明显放量
- 量比 > 5：放量明显，可能有机会
- 量比 < 1：缩量

---

### 3. 股价

**选股标准：**

| 市值类型 | 股价范围 | 说明 |
|---------|---------|------|
| 小盘股 | 10-30元 | 弹性大 |
| 中盘股 | 30-100元 | 适中 |
| 大盘股 | 100元以上 | 稳健 |

**实战应用：**
- 低价股弹性大，但可能有基本面问题
- 中价股是游资最爱
- 高价股机构为主

---

### 4. 涨停天数

**定义：** 连续涨停的天数

**选股标准：**

| 连板数 | 阶段 | 策略 |
|-------|------|------|
| 1板 | 启动 | 观察 |
| 2板 | 发酵 | 重点关注 |
| 3板 | 确认 | 可买入 |
| 4板+ | 主升 | 龙头可追 |
| 7板+ | 高潮 | 谨慎 |

**实战应用：**
- 首板：需要观察板块效应
- 二板：确认板块强度
- 三板以上：龙头确认

---

### 5. 振幅

**定义：** 当日最高价与最低价之差/昨日收盘价

**选股标准：**

| 振幅 | 含义 |
|-----|------|
| < 3% | 窄幅震荡 |
| 3-7% | 正常波动 |
| 7-10% | 波动较大 |
| > 10% | 剧烈波动 |

**实战应用：**
- 振幅适中（5-10%）的股票有操作空间
- 振幅过小（<3%）弹性差
- 振幅过大（>15%）风险高

---

### 6. 均线

**常用均线：** 5日、10日、20日、30日、60日、120日、250日

**选股标准：**

| 均线形态 | 含义 |
|---------|------|
| 多头排列 | 上涨趋势 |
| 空头排列 | 下跌趋势 |
| 均线缠绕 | 震荡整理 |

**实战应用：**
- 5日线上持有
- 10日线附近低吸
- 20日线止损
- 60日线趋势线

---

## 三、综合选股条件

### 1. 首板选股条件

```
换手率 > 3%
量比 > 1.3
涨幅 3-7%
股价 < 100元
流通市值 50-300亿
板块有涨停效应
```

### 2. 接力选股条件

```
换手率 > 5%
量比 > 2
二连板及以上
板块效应明显
市场最高板
```

### 3. 低吸选股条件

```
回调到10日线
缩量企稳
量比 < 1
均线支撑有效
```

## 四、不同策略的量化条件

### 1. 打板策略
- 换手率：3-10%
- 量比：1-3
- 涨幅：9.5%以上（涨停价）
- 封板时间：越早越好
- 封板质量：无爆量

### 2. 低吸策略
- 换手率：1-3%
- 量比：<1（缩量）
- 回调位置：5日/10日/20日均线
- 支撑位：明显

### 3. 首板策略
- 换手率：>3%
- 量比：>1.3
- 涨幅：3-7%
- 流通市值：50-300亿
- 板块效应：有

---

## 五、量化选股流程

### 1. 初选
- 换手率 > 3%
- 量比 > 1
- 非ST股
- 非退市风险

### 2. 精选
- 板块有涨停效应
- 走势形态良好
- 有题材支撑

### 3. 优选
- 龙头优先
- 换手充分
- 量价配合

---

## 六、注意事项

1. **指标是参考**：不能单独依靠某一个指标
2. **结合情绪**：量化条件+情绪判断
3. **结合板块**：必须在主流板块
4. **结合形态**：技术形态要配合
5. **动态调整**：根据市场环境调整参数

---

## 七、常见量化陷阱

1. **过度拟合**：参数过于精确反而无效
2. **忽略情绪**：量化必须结合情绪周期
3. **忽视板块**：必须主流板块
4. **静态参数**：市场在变，参数也要变
5. **幸存者偏差**：只看成功的案例
