# 东方财富拉萨团结路第二证券营业部调研报告

## 一、游资简介和背景

### 1.1 基本信息

- **营业部全称**：东方财富证券股份有限公司拉萨团结路第二证券营业部
- **曾用名**：西藏东方财富证券股份有限公司拉萨团结路第二证券营业部（2015年-2018年）
- **成立时间**：2015年6月2日
- **注册地址**：西藏拉萨市柳梧新区国际总部城10栋1单元1层1号102室
- **公司性质**：东方财富证券股份有限公司分公司（非上市）

### 1.2 背景起源

拉萨团结路第二营业部并非传统意义上的游资席位，而是一个**散户大本营**。其形成背景如下：

1. **历史渊源**：2015年，东方财富信息股份有限公司收购西藏同信证券股份有限公司，从而获得了证券从业资格，随后更名为东方财富证券股份有限公司。

2. **互联网开户聚集效应**：东方财富凭借其强大的互联网金融平台优势和影响力，吸引大量散户投资者在网上开户。这些客户的账户被自动分配到东方财富在西藏的营业部，其中最主要的四个营业部被市场合称为**"拉萨天团"**或**"四大名部"**：
   - 拉萨团结路第一证券营业部
   - 拉萨团结路第二证券营业部
   - 拉萨东环路第一证券营业部
   - 拉萨东环路第二证券营业部

3. **散户集中营**：这些营业部聚集了大量散户投资者，因此他们的买卖行为会以营业部名义出现在龙虎榜中，形成了独特的"拉萨天团"现象。

## 二、交易风格和策略特点

### 2.1 核心风格特征

| 特点 | 描述 |
|------|------|
| **操作周期** | 超短线为主，通常1-4天 |
| **主要策略** | 热门票反复做T、高抛低吸 |
| **买入方式** | 擅长水下低吸、大阴线买入 |
| **出货方式** | 一日游为主，次日及时了结 |
| **风险控制** | 被套会认赔割肉，绝不恋战 |

### 2.2 策略详解

1. **T+0操作模式**：
   - 在同一交易日内买入和卖出股票
   - 通过反复做T来降低成本、增加利润
   - 利用市场震荡寻找拉升机会

2. **低吸策略**：
   - 擅长大阴线买入：在股价大幅下跌时买入"抄底"
   - 水下低吸：在股价在水下运行时进行低吸
   - 如果能低吸到"大长腿"（即从低位大幅拉升），当日收益可能达到15%

3. **追涨杀跌**：
   - 当股票上涨时跟风买入
   - 当股票下跌时恐慌性抛售
   - 这种操作风格加剧了股价波动

4. **与其他游资的博弈**：
   - 经常与知名游资（如赵老哥）进行对手盘操作
   - 游资拉高时拉萨天团卖出，游资砸盘时拉萨天团买入
   - 高抛低吸技巧娴熟

## 三、典型的操作手法

### 3.1 主要手法分类

**手法一：跟游资"埋坑"**

- 游资拉涨停时，拉萨天团会趁机出货
- 游资砸盘时，拉萨天团会反向买入
- 这种操作经常让其他游资"被埋"

**手法二：涨停板接力**

- 当拉萨天团封涨停后，第二天连板概率较高
- 经常采用顶一字板或高开秒板的方式
- 吸引其他游资追高入场，然后集体出货

**手法三：震荡市低吸**

- 在市场震荡时，在股价下跌过程中以大阴线买入
- 利用市场波动寻找拉升机会
- 通过反复做T来获取利润

**手法四：抱团操作**

- 多个拉萨营业部同时出现在龙虎榜买卖席位
- 展现出较强的协同性和凝聚力
- 经常包揽买卖前五席位

### 3.2 操作特点总结

1. **分散性**：虽然资金体量较大，但由众多散户组成，缺乏统一组织
2. **高频性**：交易频繁上榜，根据2021年数据，该营业部全年上榜5231次
3. **灵活性**：操作风格灵活多变，善于适应不同市场环境
4. **影响大**：由于上榜次数频繁，对股价的短期影响显著

## 五、操作股票的类型和偏好

### 5.1 题材偏好

根据2024年数据和市场分析，拉萨天团（包括团结路第二营业部）的偏好题材包括：

| 偏好类型 | 说明 |
|----------|------|
| **汽车股** | 偏好汽车整车及零部件板块 |
| **计算机软件** | 偏好软件类科技股 |
| **通用设备** | 制造业相关标的 |
| **北交所个股** | 对北交所上市股票兴趣浓厚 |
| **次新股** | 偏好刚上市不久的股票 |
| **热门概念** | 追逐市场热点题材（5G、科创板、东数西算、DeepSeek等） |

### 5.2 股票特征

拉萨天团操作的股票通常具有以下特征：

1. **高换手率**：日换手率经常超过20%
2. **高振幅**：日内波动幅度大
3. **热门题材**：涉及市场当前热点概念
4. **流动性好**：成交金额较大，方便进出
5. **波动性大**：股价经常出现"地天板"或"天地板"走势

## 六、总结与评价

### 6.1 营业部特点总结

东方财富证券拉萨团结路第二证券营业部是一个极具特色的游资席位：

1. **本质是散户集中营**：由大量通过东方财富互联网平台开户的散户组成
2. **上榜频率极高**：是龙虎榜的"常客"，成交金额长期位居前列
3. **操作风格鲜明**：超短线、高频交易、追涨杀跌
4. **市场影响力大**：操作风格激进，经常导致股价剧烈波动

### 6.2 投资参考价值

对于普通投资者而言：

- **风险警示**：拉萨天团出现后股价经常大幅波动，追高需谨慎
- **反向指标**：部分投资者将其视为反向指标（"拉萨进场，游资离场"）
- **参考学习**：其操作手法可作为短线交易的学习案例
- **风险规避**：看到拉萨天团出现在龙虎榜时，应注意风险控制
