## 二、详细流程节点

### 1. 盘前：情绪周期判断 (09:15)

> **游资核心：情绪周期是短线交易的核心，决定是否出手、仓位多少**

```
输入                              输出
─────                            ────
• 昨日涨停板数据                  • 情绪周期阶段
• 昨日炸板数据                     - 冰点/启动/发酵/高潮/退潮/混沌
• 板块资金流向                   • 仓位建议
• 大盘指数位置(20日线)           - 冰点: 0-1成
• 赚钱/亏钱效应指标               - 启动: 3-5成
• 空间板高度                      - 发酵: 5-7成
                                  - 高潮: 5-7成(逐步减)
                                  - 退潮: 0-1成
                                  - 混沌: 1-2成
```

### 2. 盘前：仓位策略确定

> **游资核心：根据情绪周期确定仓位，赢面决定仓位**

```
情绪周期         仓位建议        赢面阈值
────────────────────────────────────────────
冰点期           0-1成          不做
混沌期           1-2成          <70%
启动期           3-5成          60%-70%
发酵期           5-7成          70%-80%
高潮期           5-7成          80%-90%(逐步减)
退潮期           0-1成          不做
```

### 3. 盘前：AI生成交易计划

> **游资核心：只做龙头、不符合模式不做、严格止损**

```
输入                              输出
─────                            ────
• 情绪周期分析结果                • 交易计划列表
• 仓位策略                       每条计划包含:
• 用户策略配置                     - 股票代码+名称
• 账户资金状况                     - 买入方式(竞价/低吸/打板/半路/尾盘)
• 目标股池                        - 目标价/仓位/止损/止盈验证条件(量价/换手/板块)
                                  - 信心指数(AI评估)
                                  - 审批状态
                                  - 时间止损(隔日30分钟)

                                  - ```

### 4. 盘前/盘中：用户审核

```
权限模式           流程
─────────         ────
auto(自动)        AI计划 → 直接进入执行队列
confirm(确认)     AI计划 → 推送用户 → 用户批准/修改/拒绝 → 执行
manual(手动)      AI计划 → 仅作建议 → 用户手动执行
```

### 5. 盘中：实时监控执行

> **游资核心：时间止损 > 价格止损，流动性止损 > 技术止损**

```
触发类型              判断逻辑                    动作
──────────────────  ────────────────────────   ──────────────────
价格触发              触及目标价 + 条件验证       执行/等待/跳过
时间止损              隔日30分钟未达预期强度       立即砍仓
流动性止损            买一档位堆积量骤减80%       不计成本出货
环境止损              天地板/跌停闷杀≥3家        清仓观望
止损预警              亏损-5%                    强提醒
触及止损              亏损-5%~-7%                自动执行卖出
```

### 6. 盘中：止盈执行

> **游资核心：达到预期就卖、龙头断板必卖、移动止损保护利润**

```
止盈条件               触发条件                   动作
───────────────────  ──────────────────────    ───────────────────
固定止盈              盈利+7%~+15%              卖出1/2
移动止损              盈利+5%后                 止损上移到成本价
均线止盈              跌破5日/10日均线           卖出
龙头断板              总龙头断板                 第一时间卖出
情绪退潮              跌停板增多/炸板率上升       卖出
```

### 7. 盘后：AI复盘学习

> **游资核心：错了就要认，认了就要改，持续进化**

```
复盘维度                           输出
──────────                        ────
• 交易绩效统计                     • 胜率/盈亏比/计划完成率
• 买卖点分析                     • 模式胜率排名
• 时间止损执行情况                • 优化建议
• 仓位执行情况                    • 策略调整建议
• 模式效果排名
```

## 三、游资核心要素整合

### 3.1 买入方式选择

> **五种买入方式，需根据情绪周期和个股类型选择**

```
买入方式       适用场景                    情绪周期匹配
────────────────────────────────────────────────────────────
竞价买入       情绪启动期、龙头一字板       启动期
低吸回调       龙头首阴、回调5/10日线      发酵期/高潮期
打板/封板     情绪发酵/高潮、龙头确认板    发酵期/高潮期
半路板        确认会封板时                发酵期
尾盘买入      行情不确定时防守             退潮期/混沌期
```

### 3.2 选股标准

> **游资核心：只做龙头、不做杂毛**

```
选股优先级         标准
───────────────────────────────
龙头股            1. 板块涨停最多 2. 最早封板 3. 成交额最大
补涨股            1. 与龙头属性相同 2. 涨幅落后 3. 有补涨需求
首板股            1. 竞价强势 2. 题材新颖 3. 板块效应
```

### 3.3 风控体系

> **游资核心：宁可少赚，也要控制回撤**

```
风控优先级         说明
────────────────────────────────
时间止损           > 价格止损(隔日30分钟未达预期立即砍)
流动性止损         > 技术止损(封单骤减80%立即走)
环境止损           > 个股止损(市场天地板≥3时空仓)
固定止损           单笔-5%~-7%，日累计-5%，周-10%
```

### 3.4 仓位管理

> **游资核心：只做确定性机会，没有把握时空仓**

```
仓位原则                数值
──────────────────────────────
单票最大仓位             ≤30%
单日最大仓位             ≤70%
单笔风险敞口             ≤账户2%
日累计亏损限额           ≤5%
周最大回撤               ≤10%
```

## 五、数据检查清单

### 5.1 买入前检查

```
情绪条件
[ ] 当前情绪周期适合买入(启动/发酵/高潮)
[ ] 赚钱效应良好
[ ] 无明显亏钱效应

大盘条件
[ ] 大盘无系统性风险
[ ] 大盘在20日线之上

板块条件
[ ] 有明确的主流板块
[ ] 板块效应明显

个股条件
[ ] 是板块龙头/最强股
[ ] 涨停质量好(量价配合)
[ ] 符合买入模式
```

### 5.2 持仓中检查

```
[ ] 是否到止损位
[ ] 是否到时间止损(隔日30分钟)
[ ] 环境是否恶化(天地板≥3)
[ ] 流动性是否充足(封单骤减)
[ ] 是否出现止盈信号
[ ] 龙头是否断板
```

### 5.3 卖出后检查

```
[ ] 是否执行了止损/止盈
[ ] 仓位是否需要调整
[ ] 日亏损是否超5%
[ ] 是否需要总结经验
```

## 六、与现有系统的对比

| 维度 | 现有系统 | AI化游资系统 |
|------|----------|--------------|
| 情绪判断 | 用户自己看 | AI自动判断+阶段输出 |
| 仓位策略 | 用户自己定 | AI根据情绪周期推荐 |
| 选股 | 用户自己选 | AI按龙头标准筛选 |
| 买入方式 | 用户自己选 | AI根据周期匹配 |
| 止损 | 用户自己执行 | AI自动执行(时间/价格/流动性) |
| 止盈 | 用户自己定 | AI移动止损+分批卖出 |
| 盘后复盘 | 用户自己总结 | AI自动统计+模式学习 |
| 时间投入 | 高(全程手动) | 中(审核+干预) |
| 执行纪律 | 靠人性 | 靠系统强制 |

## 七、游资心法整合

### 7.1 核心纪律

```
买入纪律
1. 不符合条件不开仓
2. 没有确定性不做
3. 只做模式内交易
4. 控制仓位

卖出纪律
1. 止损必须坚决
2. 止盈可以分批
3. 龙头断板必卖
4. 退潮必须跑
```

### 7.2 养家心法仓位法

```
赢面60%以下     →  观望
赢面60%-70%    →  小仓出击
赢面70%-80%    →  中仓出击
赢面80%-90%    →  大仓出击
赢面90%以上    →  满仓
```

### 7.3 顶级游资铁律

```
1. "错了就要认，认了就要改"
2. "宁可少赚，也要控制回撤"
3. "止损要坚决，不能犹豫"
4. "只做龙头，不做杂毛"
5. "退潮必须跑，不格局"
```
