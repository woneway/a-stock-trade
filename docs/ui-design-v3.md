## 五、总结

**核心流程**：复盘 → 选股 → 计划 → 监控 → 记录

| 步骤 | 页面 | 状态 |
|------|------|------|
| 1 | 复盘 | 分析昨日市场，判断情绪周期 |
| 2 | 选股 | 筛选符合条件的股票 |
| 3 | 计划 | 制定买入/卖出计划 |
| 4 | 监控 | 实时盯盘，执行计划 |
| 5 | 记录 | 记录成交，统计绩效 |

每个页面都有明确的**下一步**指引，形成闭环。
