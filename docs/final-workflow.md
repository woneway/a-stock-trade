# 游资交易系统 - 最终版工作流设计

> 基于严谨分析的工作流设计，聚焦实用性与可执行性

---

## 一、核心工作流

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           每日工作流                                   │
└─────────────────────────────────────────────────────────────────────────────┘

15:00-17:00  【盘后复盘】
│   └── 今日市场分析（客观）
│       ├── 基础数据分析
│       ├── 情绪周期判断
│       ├── 板块分析
│       └── 风险警示
│
09:00-09:15  【开盘前准备】
│   └── 制定今日操作计划（主观）
│       ├── 情绪周期确认
│       ├── 仓位确认
│       ├── 关注股票
│       └── 买入条件
│
09:30-11:30  【上午盘】
│   └── 执行 + 记录
│       ├── 执行买入/卖出
│       ├── 10:00时间止损检查 ⚡
│       └── 记录成交
│
11:30-13:00  【午间休息】
│   └── 上午复盘 + 下午准备
│
13:00-14:30  【下午盘】
│   └── 执行 + 记录
│
14:30-15:00  【尾盘处理】
│   └── 移动止损 + 明日计划
│
15:00+  【盘后记录】
│   └── 今日操作总结
```

**核心原则**：
- 盘后复盘 = 分析今天（客观）
- 开盘前计划 = 决定今天怎么做（主观）
- 盘中执行 = 快速操作（行动）
- 盘后记录 = 总结今天（反馈）

## 二、盘后复盘阶段（15:00-17:00）

> 核心问题：今天市场怎么样？

### 2.1 今日市场分析

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  今日市场分析 - 盘后15:00-17:00                                       │
└─────────────────────────────────────────────────────────────────────────────┘

1. 基础数据（必须记录）
   ├── 涨停数量: ___家 (>30正常, <10冰点)
   ├── 跌停数量: ___家 (<10正常, >20退潮)
   ├── 炸板率: ___% (<25%正常, >40%危险)
   ├── 最高连板: ___板
   ├── 上涨家数: ___
   └── 成交额: ___亿

2. 大盘位置
   ├── 大盘在20日线之上: [是/否]
   ├── 指数趋势: [上涨/下跌/震荡]
   └── 黄白线关系: [黄上白下/黄下白上/纠缠]

3. 情绪周期判断（核心！）
   ├── 当前周期: [冰点/启动/发酵/高潮/分歧/退潮/混沌]
   ├── 判断依据:
   │   ├── 涨停数量变化
   │   ├── 炸板率变化
   │   ├── 最高板变化
   │   └── 大盘位置
   │
   └── 仓位建议:
       ├── 冰点: 0-1成（空仓）
       ├── 混沌: 1-2成
       ├── 启动: 3-5成
       ├── 发酵: 5-7成
       ├── 高潮: 5-7成（逐步减）
       └── 退潮: 0-1成

4. 板块分析
   ├── 主流板块: [板块1, 板块2]
   ├── 支线板块: [板块3]
   ├── 涨停数: ___
   ├── 龙头股: ___
   └── 持续性: [持续/分化/消退]

5. 风险警示
   ├── 风险信号: [有/无]
   └── 风险描述: ___
```

### 2.2 输出：复盘记录

```
复盘记录包含：
- 基础数据
- 大盘位置
- 情绪周期 + 仓位建议
- 热门板块 + 龙头股
- 风险警示
```

## 四、盘中执行阶段（09:30-15:00）

> 核心问题：买不买？卖不卖？

### 4.1 买入决策（简化版）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  买入决策 - 简化3问                                                    │
└─────────────────────────────────────────────────────────────────────────────┘

问题1: 这只股票在计划内吗？
       ├── 是 → 继续
       └── 否 → 不做

问题2: 价格到目标了吗？
       ├── 是 → 继续
       └── 否 → 等待

问题3: 验证条件满足吗？
       ├── 是 → 执行买入
       └── 否 → 放弃
```

**说明**：盘前已经确认过情绪、大盘、板块、龙头等条件，盘中只需要快速确认这3点。

### 4.2 买入条件详情（可展开查看）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  买入条件详情（可展开）                                                │
└─────────────────────────────────────────────────────────────────────────────┘

情绪条件: 启动/发酵/高潮才买
大盘条件: 在20日线之上，无系统风险
板块条件: 有主流板块效应
个股条件: 是板块龙头/最强
模式条件:
  ├── 竞价: 9:15-9:25，量价匹配
  ├── 低吸: 回调到均线，缩量企稳
  ├── 半路: 冲击涨停，分时流畅
  ├── 打板: 涨停瞬间，板块效应强
  └── 尾盘: 14:30后，涨幅<5%
```

### 4.3 卖出决策

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  卖出决策 - 4种情况                                                   │
└─────────────────────────────────────────────────────────────────────────────┘

情况1: 触及止损 ⚡
   ├── 条件: 亏损达到 -5% ~ -7%
   ├── 动作: 立即卖出，不犹豫
   └── 记录: 止损

情况2: 触及止盈
   ├── 条件1: 盈利达到 +7% → 卖出1/2，其余移动止损
   ├── 条件2: 盈利达到 +15% → 再卖出1/2
   └── 记录: 止盈1/止盈2

情况3: 移动止损
   ├── 条件: 盈利>5%后跌破成本价
   ├── 动作: 立即卖出
   └── 记录: 移动止损

情况4: 时间止损 ⚡⚡（游资核心！）
   ├── 条件: 隔日开盘30分钟后未达预期强度
   ├── 动作: 立即砍仓，不犹豫
   └── 记录: 时间止损
```

### 4.4 盘中检查清单

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  盘中检查清单 - 关键时间点                                            │
└─────────────────────────────────────────────────────────────────────────────┘

09:30-10:00 ⚡ 时间止损检查
   ├── 检查持仓股是否在30分钟内达到预期强度
   ├── 未达预期 → 立即卖出（时间止损）
   └── 这是游资核心纪律！

10:00 环境检查
   ├── 检查天地板数量
   ├── ≥3家 → 建议空仓/减仓
   └── 记录: 环境止损

11:30 上午盘结束
   ├── 检查上午走势
   ├── 评估是否需要调整下午策略

14:30 尾盘处理
   ├── 盈利股移动止损保护利润
   ├── 亏损股符合止损条件？→ 卖出
   └── 可考虑尾盘选股
```

## 五、风控核心原则

### 5.1 风控优先级

```
时间止损 > 价格止损 > 环境止损 > 流动性止损
```

### 5.2 风控清单

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  风控原则                                                            │
└─────────────────────────────────────────────────────────────────────────────┘

1. 仓位控制
   ├── 单票不超30%
   ├── 单日不超70%
   └── 只做成交额亿以上股票

2. 回撤控制
   ├── 日亏损-5%必须停手
   ├── 周亏损-10%必须空仓
   └── 单笔风险敞口≤2%

3. 止损执行
   ├── 时间止损：隔日30分钟未达预期立即砍仓
   ├── 价格止损：-5%无条件止损
   └── 环境止损：天地板≥3家时空仓观望
```

### 5.3 游资纪律

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  核心纪律                                                            │
└─────────────────────────────────────────────────────────────────────────────┘

买入纪律：
1. 不符合情绪周期不做
2. 不符合模式不做
3. 没有确定性不做
4. 不做杂毛只做龙头
5. 大盘在20日线下不做

卖出纪律：
1. 止损必须坚决（-5%无条件）
2. 时间止损：隔日10:00前未达预期立即砍仓
3. 止盈可以分批
4. 龙头断板必卖
5. 退潮必须跑
6. 环境恶化必走
```

### 6.1 交易记录字段

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  交易记录 - 卖出原因选项                                              │
└─────────────────────────────────────────────────────────────────────────────┘

计划执行：
- 计划买入 / 计划卖出

止盈止损触发：
- 止盈1 (+7%)
- 止盈2 (+15%)
- 移动止损
- 止损 (-5%)

特殊止损：
- 时间止损（隔日30分钟未达预期）
- 环境止损（天地板≥3家）
- 流动性止损（买一骤减80%）

临时卖出：
- 龙头断板
- 板块退潮
- 非预期下跌

手动操作：
- 手动卖出
- 未执行（放弃）
```

### 6.2 复盘记录字段（第一阶段）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  复盘记录 - 第一阶段字段（现有系统兼容）                               │
└─────────────────────────────────────────────────────────────────────────────┘

必填：
- trade_date: 日期
- limit_up_count: 涨停数量
- limit_down_count: 跌停数量
- market_cycle: 情绪周期
- position_advice: 仓位建议
- hot_sectors: 热门板块

选填：
- broken_plate_ratio: 炸板率
- highest_board: 最高连板
- up_count: 上涨家数
- turnover: 成交额
- risk_warning: 风险警示

扩展（第二阶段）：
- above_20ma: 大盘是否在20日线
- main_sectors: 主流板块
- sector_stocks: 龙头股
```

## 八、核心要点总结

|------|---------|--------|
| 盘后复盘 | 分析今天市场 | 客观记录数据，判断情绪周期 |

| 盘后记 | 总结今天 | 正确+错误+改进 |
