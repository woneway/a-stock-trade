# 游资逻辑原理深度解析

> 本文档深入讲解每个操作步骤背后的逻辑原理
> 回答「为什么要这样做」而非「怎么做」

---

## 一、为什么每天都要复盘？

### 1.1 复盘的本质

**复盘 = 市场的「体检报告」**

你见过医生不检查就直接开药的吗？交易也是一样的道理。

```
不复盘 = 盲目交易 = 赌博
复盘 = 了解市场现状 = 理性交易
```

**复盘要回答的核心问题**：

1. **今天市场整体怎么样？** → 情绪判断
2. **谁在涨？为什么涨？** → 涨停板复盘
3. **谁在跌？为什么跌？** → 炸板复盘
4. **主力资金去了哪里？** → 龙虎榜+资金流向
5. **明天应该怎么做？** → 制定计划

---

### 1.2 不复盘的代价

**凭感觉交易的后果**：

| 感觉 | 实际情况 | 结果 |
|------|----------|------|
| 觉得会涨 | 可能是下跌中继 | 买在半山腰 |
| 觉得跌够了 | 可能是下跌中继 | 接飞刀 |
| 觉得是龙头 | 可能是跟风 | 涨幅小还容易被套 |
| 觉得安全 | 可能是补跌 | 大亏 |

**复盘的本质是「偷看答案」**：
- 今天的涨停股已经告诉你资金在炒什么
- 今天的炸板股已经告诉你什么不能买
- 今天的龙虎榜已经告诉你主力在哪
- 你不去看，等于考试不看题目

## 二、为什么先判断情绪再交易？

### 2.1 情绪是市场的「温度计」

**逻辑链**：

```
市场情绪 → 决定赚钱难度 → 决定仓位 → 决定策略
```

**为什么情绪这么重要？**

```
牛市（高潮期）：80%的股票都在涨，你随便买都可能赚
熊市（退潮期）：80%的股票都在跌，你买什么都可能亏
```

**情绪周期的本质**：

> 情绪周期本质是「供需关系」的周期性表现
> - 买的人多（情绪高）→ 股票供不应求 → 价格上涨
> - 卖的人多（情绪低）→ 股票供过于求 → 价格下跌

### 2.2 为什么涨停家数是情绪第一指标？

**涨停板 = 资金用真金白银投票「这个股票值得买」**

```
涨停家数多 = 大量资金在积极做多 = 强势市场
涨停家数少 = 资金都在观望 = 弱势市场
```

**涨停家数的深层含义**：

| 涨停家数 | 背后逻辑 |
|----------|----------|
| >50家 | 资金大规模入场，市场供不应求，强势 |
| 20-50家 | 资金部分入场，结构性机会，中性 |
| <20家 | 资金入场意愿低，弱势 |

### 2.3 为什么炸板率重要？

**炸板 = 资金尝试做多但失败了**

```
炸板率高 = 资金做多失败率高 = 市场环境恶劣
炸板率低 = 资金做多成功率高 = 市场环境友好
```

**炸板率的深层逻辑**：

```
炸板率40% = 你打10个板，4个会炸
炸板率15% = 你打10个板，只有1.5个会炸
打板胜率差4倍！
```

### 2.4 为什么要看最高板？

**最高板 = 市场最强资金认可的股票**

```
5板龙头 = 市场最强资金认可的标的
2板股 = 市场普通资金认可的标的
```

**最高板的意义**：

- 最高板越高，说明资金敢于进攻的空间越大
- 最高板从5板跌到2板，说明资金不敢做高了
- 最高板A杀（连续跌停），说明风险极大

## 三、为什么要关注涨停板？

### 3.1 涨停板的本质

**涨停板 = 资金抢筹的极限表现**

```
股票涨到10%就涨不动了（涨停）
说明想买的人太多，卖家不够
这就是「供不应求」
```

**涨停板是市场最强信号**：

| 信号类型 | 强度 |
|----------|------|
| 涨停板 | ★★★★★ |
| 涨幅>7% | ★★★★☆ |
| 涨幅>5% | ★★★☆☆ |
| 涨幅<3% | ★★☆☆☆ |

### 3.2 为什么要按涨停时间排序？

**越早涨停 = 越强**

逻辑很简单：

```
09:25涨停：开盘就涨停，说明资金一开盘就迫不及待要买
14:30涨停：到尾盘才涨停，说明资金是没办法才拉的
```

**时间背后的资金态度**：

| 涨停时间 | 资金态度 | 强度 |
|----------|----------|------|
| 09:25-09:30 | 开盘即抢，极度看好 | 强 |
| 09:30-10:30 | 早盘积极做多 | 较强 |
| 10:30-14:00 | 盘中跟涨 | 中 |
| 14:00-15:00 | 尾盘被迫拉升 | 弱 |

### 3.3 为什么要看封单金额？

**封单 = 涨停后还在排队等待买入的资金**

```
封单大（10亿）+ 封单稳定 = 资金强烈看好，还会涨
封单小（100万）+ 封单减少 = 资金不看好，随时可能开板
```

### 3.4 为什么要区分龙头和跟风？

**核心逻辑：龙头的溢价来自「领涨」，跟风的溢价来自「被带动」**

```
龙头涨停 → 带动板块 → 跟风跟着涨
跟风涨停 → 只是跟随 → 没有带动性
```

**龙头vs跟风的本质区别**：

| 属性 | 龙头 | 跟风 |
|------|------|------|
| 定位 | 定价者 | 跟随者 |
| 溢价 | 主动溢价 | 被动溢价 |
| 持续性 | 强 | 弱 |
| 风险 | 较低 | 高 |

## 四、为什么要复盘炸板股？

### 4.1 炸板的本质

**炸板 = 资金做多失败**

```
本来看好要涨停 → 结果没封住 → 资金跑了
```

**炸板后会发生什么？**

```
炸板 → 当天回落 → 第二天低开 → 可能继续跌
```

### 4.2 为什么要分析炸板原因？

**不同原因，后续走势完全不同**

| 炸板类型 | 原因 | 后续走势 |
|----------|------|----------|
| 跟风炸板 | 跟风股没人要 | 大概率继续跌 |
| 龙头炸板 | 获利盘卖出 | 可能反包 |
| 板块炸板 | 整个板块不行 | 规避整个板块 |
| 量能不足 | 资金不够 | 观察量价 |

### 4.3 跟风炸板为什么不能买？

**跟风炸板的逻辑**：

```
1. 龙头涨停 → 跟风涨停
2. 龙头没封住 → 跟风先炸
3. 跟风炸了说明：资金不认可这个股票
4. 第二天低开是大概率事件
```

**统计数据**：
- 跟风炸板：70%以上会低开
- 龙头炸板：40%左右会反包

## 五、为什么要看龙虎榜？

### 5.1 龙虎榜的本质

**龙虎榜 = 主力资金动向的「监控摄像头」**

```
散户买卖：几百几千股，对股价影响微乎其微
主力买卖：几万几十万股，直接决定股价涨跌
跟踪主力 = 站在资金这一边
```

### 5.2 为什么要区分机构和游资？

**不同资金，操作风格完全不同**

| 资金类型 | 操作风格 | 持有时间 | 拉升方式 |
|----------|----------|----------|----------|
| 机构资金 | 价值投资 | 几周-几个月 | 缓慢推升 |
| 游资资金 | 趋势投机 | 几天-几周 | 快速拉升 |

**为什么机构+游资合力最好？**

```
机构买入 = 基本面认可，长期看好
游资买入 = 技术面认可，短期要涨
两者合力 = 既要涨，又要涨得久
```

### 5.3 为什么要避免「对倒」？

**对倒 = 左手卖右手，制造虚假成交量**

```
目的：吸引散户跟进，然后出货
结果：散户追高被套
```

**对倒的特征**：

```
买入 ≈ 卖出
今天买，明天卖
营业部反复出现
```

## 六、为什么要看资金流向？

### 6.1 资金是股价涨跌的根本

**股价上涨的唯一原因：钱多**

```
供需关系：
买的人多（资金流入）→ 供不应求 → 价格上涨
卖的人多（资金流出）→ 供过于求 → 价格下跌
```

**资金流向是领先指标**：

```
基本面变化 → 资金发现 → 资金流入 → 股价上涨
技术面变化 → 资金发现 → 资金流入 → 股价上涨
消息面变化 → 资金发现 → 资金流入 → 股价上涨
```

### 6.2 为什么要用区间统计？

**单日资金流入可能是假象**

```
今天资金流入 = 有可能明天就跑
连续5天资金流入 = 大概率是真的要涨
```

**区间统计的意义**：

```
单日净流入：脉冲行情，持续性不确定
5日净流入：资金持续关注，可信度高
10日净流入：资金深度布局，大行情
```

### 6.3 为什么要找持续净流入？

**持续净流入 = 主力真金白银在建仓**

```
逻辑链：
持续净流入 → 主力在吸筹 → 后面要拉升 → 现在是低位 → 应该买
```

## 七、为什么要区分周期？

### 7.1 周期决定策略

**不同周期，赚钱难度完全不同**

| 周期 | 赚钱难度 | 正确策略 |
|------|----------|----------|
| 启动期 | 中 | 试错龙头 |
| 发酵期 | 易 | 积极参与 |
| 高潮期 | 易 | 持股待涨 |
| 分歧期 | 难 | 谨慎追高 |
| 退潮期 | 极难 | 空仓休息 |
| 冰点期 | 极难 | 等待 |

### 7.2 周期背后的原理

**周期的本质是「情绪的轮回」**

```
启动：少数人发现机会 → 开始买
发酵：更多人发现机会 → 抢着买
高潮：所有人都在买 → 买无可买
分歧：有人开始害怕 → 获利了结
退潮：越来越多的人卖 → 跌跌不休
冰点：没人敢买了 → 极度悲观
```

### 7.3 周期与仓位的关系

**为什么退潮期要空仓？**

```
退潮期特征：
- 涨停少（机会少）
- 炸板多（成功率低）
- 跌停多（风险大）
- 龙头A杀（亏起来很凶）

概率视角：
高潮期买股：70%赚钱
退潮期买股：70%亏钱
```

## 八、为什么要设止损？

### 8.1 止损的本质

**止损 = 承认错误 + 截断亏损**

```
你买入一只股票
期望它涨
但它跌了
说明你的判断可能是错的
止损 = 承认错误，纠正错误
```

### 8.2 为什么要用-5%止损？

**数学原理：亏50%需要赚100%才能回本**

```
亏5%：赚5.26%回本
亏10%：赚11.11%回本
亏20%：赚25%回本
亏50%：赚100%回本
亏70%：赚233%回本
```

**小亏的目的**：
- 保护本金
- 保持交易主动性
- 避免深套

### 8.3 为什么要结合大盘止损？

**大盘跌，个股很难独善其身**

```
大盘跌2%：
- 90%的股票会跌
- 只有10%的股票能涨
- 你凭什么是那10%？
```

**大盘急跌时的策略**：
- 大盘跌>2%：停止开新仓
- 大盘跌>3%：考虑减仓
- 大盘跌>5%：空仓观望

## 九、为什么要看竞价？

### 9.1 竞价的本质

**竞价 = 当天第一轮「资金投票」**

```
9:15-9:25的资金态度 = 当天开盘的预期
高开：资金看好
低开：资金不看好
```

### 9.2 竞价为什么重要？

**竞价是「盘前预演」**

```
开盘半小时的走势 = 竞价结果的延续
竞价高开高走 → 强势延续
竞价低开低走 → 弱势延续
```

### 9.3 为什么要看竞价量能？

**量能 = 资金的参与程度**

```
竞价量能大（比昨日放大）：
- 说明资金参与积极
- 抢筹/抢跑的态度坚决
- 信号可信度高

竞价量能小：
- 说明资金参与度低
- 观望情绪浓厚
- 信号可信度低
```

## 十、为什么要分仓？

### 10.1 分仓的本质

**分仓 = 分散风险 + 保持主动**

```
单一持仓风险：
- 单股跌停 → 账户大伤
- 单股停牌 → 资金冻结

分仓优势：
- 单股跌停 → 伤害有限
- 保持流动性 → 随时可以调仓
```

### 10.2 仓位的本质

**仓位 = 你对这次机会的「下注比例」**

```
机会确定（高潮期）→ 重仓下注
机会不确定（分歧期）→ 轻仓试错
没机会（退潮期）→ 不下注
```

## 十一、实战逻辑链汇总

### 11.1 完整决策链条

```
1. 情绪判断（今天市场怎么样？）
   ↓
   涨停家数多？→ 情绪好 → 可以做
   涨停家数少？→ 情绪差 → 少做/不做

2. 周期定位（现在是什么阶段？）
   ↓
   启动/发酵/高潮 → 积极做
   分歧/退潮/冰点 → 谨慎做

3. 板块选择（资金在哪里？）
   ↓
   资金净流入板块 → 跟进
   资金净流出板块 → 回避

4. 个股选择（买什么？）
   ↓
   板块龙头 → 首选
   跟风首板 → 次选

5. 时机选择（什么时候买？）
   ↓
   竞价符合预期 → 买入
   盘中突破 → 买入

6. 风险管理（买多少？什么时候卖？）
   ↓
   设止损 → -5%
   分仓 → 单股不超2成
```

### 11.2 核心理念

**永远站在资金这一边**：
- 资金流入 → 跟进
- 资金流出 → 回避

**永远尊重市场**：
- 市场强 → 做
- 市场弱 → 不做

**永远保护本金**：
- 止损纪律
- 分仓原则
- 不借钱炒股

## 十二、常见问题解答

### Q1：为什么我复盘了还是亏钱？

可能原因：
1. 复盘了但没按计划执行
2. 计划制定了但没设止损
3. 选股选对了但买错了时机
4. 买到龙头了但没拿住

### Q2：复盘要花多长时间？

建议：
- 基础复盘：30分钟
- 深度复盘：60-90分钟
- 不要超过2小时（效率会下降）

### Q3：每天复盘有用吗？

绝对有用：
- 不复盘 = 盲人骑瞎马
- 复盘 = 考试看题目
- 坚持复盘，水平会螺旋上升

### Q4：需要记笔记吗？

非常建议：
- 记录每日的判断和结果
- 记录错误的反思
- 定期回顾总结

## 十三、总结：底层逻辑

```
市场是由「资金」驱动的
资金是由「情绪」控制的
情绪是有「周期」规律的
周期是可以通过「数据」观察的

复盘 → 观察数据 → 判断周期 → 跟随资金 → 尊重市场
```

**核心口诀**：
- 资金在哪我在哪
- 情绪好我就做
- 情绪差我就走
- 错了就止损
