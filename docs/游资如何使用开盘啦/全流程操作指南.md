# 游资开盘啦使用流程指南

> 从复盘到计划到执行，全流程开盘啦功能使用详解
> 每个阶段明确：做什么 → 用什么功能 → 怎么用 → 判断标准

---

## 一、复盘阶段（T日 15:00-18:00）

### 1.1 第一步：情绪判断（5分钟）

**任务**：快速判断今日市场整体状态

**使用功能**：

| 功能 | 位置 | 看什么 |
|------|------|--------|
| 情绪指标 | 首页顶部/情绪大师 | 涨停家数、炸板率、跌停家数 |
| 昨日涨停 | 情绪大师详情页 | 昨日涨停溢价 |
| 昨日连板 | 情绪大师详情页 | 昨日连板表现 |

**操作步骤**：

```
1. 打开开盘啦 → 首页顶部查看情绪仪表盘
2. 记录关键数据：
   - 涨停家数：___家（强>50/中20-50/弱<20）
   - 最高板：___板
   - 炸板率：___%（高>30%/中15-30%/低<15%）
   - 跌停家数：___家
3. 判断当前周期：启动/发酵/高潮/分歧/退潮/冰点
```

**判断标准**（来自官网）：

| 周期 | 涨停家数 | 最高板 | 炸板率 | 策略 |
|------|----------|--------|--------|------|
| 启动 | 20-30 | 3板 | 25-35% | 试错龙头 |
| 发酵 | 30-50 | 4-5板 | 15-25% | 积极参与 |
| 高潮 | 50+ | 6板+ | 10-15% | 持股待涨 |
| 分歧 | 30-50 | 4-5板 | 25-35% | 谨慎追高 |
| 退潮 | 10-20 | 2-3板 | 35-45% | 空仓休息 |
| 冰点 | <10 | 1-2板 | >45% | 等待新周期 |

---

### 1.2 第二步：涨停板复盘（20分钟）

**任务**：识别今日最强股票，找到明日潜在标的

**使用功能**：涨停板分析

**操作步骤**：

```
1. 首页 → 点击「涨停板」
2. 筛选「全部」→ 按「涨停时间」排序
3. 逐个查看涨停股，记录以下信息：
   - 涨停时间（越早越强）
   - 封单金额（越大越强）
   - 流通市值（越小越容易炒）
   - 涨停原因（新题材优于老题材）
4. 筛选「连板池」→ 识别龙头
   - 5板以上：市场核心
   - 3-4板：跟风龙头
   - 断板股：可能反包
```

**核心指标解读**：

| 指标 | 强势信号 | 弱势信号 |
|------|----------|----------|
| 涨停时间 | 09:25-09:30 | 14:00之后 |
| 封单金额 | >1亿 | <1000万 |
| 换手率 | 3-10% | >20%或<1% |
| 涨停原因 | 新题材/政策利好 | 跟风/老题材 |

**输出**：选出3-5只明日重点关注股

---

### 1.3 第三步：炸板复盘（10分钟）

**任务**：总结失败原因，避免重复犯错

**使用功能**：炸板分析

**操作步骤**：

```
1. 首页 → 点击「炸板分析」
2. 查看炸板股列表
3. 记录每只炸板股：
   - 炸板时间（越早炸越弱）
   - 炸板次数（多次炸极弱）
   - 回落幅度（越大越弱）
4. 分析炸板原因分类：
   - 跟风炸板（风险高，放弃）
   - 板块回调（规避板块）
   - 量能不足（观察）
   - 大盘拖累（系统性风险）
```

**炸板原因分类**：

| 类型 | 特征 | 风险等级 | 处理 |
|------|------|----------|------|
| 跟风炸板 | 龙头已封，跟风被炸 | 高 | 放弃 |
| 板块炸板 | 板块整体回落 | 高 | 规避板块 |
| 量能不足 | 封板资金不够 | 中 | 观察 |
| 大盘拖累 | 大盘大跌拖累 | 中 | 等待修复 |

**反思问题**：
- 今天打板为什么炸？
- 是选的股不行，还是买的时机不对？
- 有什么经验教训？

---

### 1.4 第四步：龙虎榜复盘（15分钟）

**任务**：追踪主力资金动向，识别合力票

**使用功能**：龙虎榜

**操作步骤**：

```
1. 首页 → 点击「龙虎榜」
2. 筛选「机构买入」→ 记录净买入前5
3. 筛选「游资买入」→ 记录知名游资动向
4. 对比分析：
   - 机构+游资合力：最佳标的
   - 纯机构买入：可能走趋势
   - 纯游资买入：短线强势
   - 对倒（买入≈卖出）：可能出货
```

**知名游资席位**（部分）：
- 华鑫证券上海宛平南路
- 东方财富拉萨团结路
- 中信证券上海溧阳路
- 国泰君安上海江苏路

**判断标准**：

| 榜单类型 | 信号 | 操作建议 |
|----------|------|----------|
| 机构大幅买入 | 看好基本面 | 可中线 |
| 游资合力买入 | 短线强势 | 短期关注 |
| 游资对倒 | 疑似出货 | 规避 |
| 分歧 | 观点不一 | 观望 |

---

### 1.5 第五步：资金流向复盘（15分钟）

**任务**：判断板块趋势，把握主线热点

**使用功能**：资金流向 → 区间统计

**操作步骤**：

```
1. 首页 → 点击「资金流向」
2. 切换「板块」→ 查看主力净流入
3. 记录净流入前5板块
4. 切换「个股」→ 查看净流入前10
5. 使用「区间统计」功能：
   - 选择区间：近5日/近10日
   - 查看持续净流入板块（主线）
   - 查看首次净流入板块（启动）
```

**区间统计使用方法**（来自官网）：

```
1. 资金流向 → 区间统计
2. 选择时间区间（如12/17-12/24）
3. 查看维度：
   - 区间涨幅：找领涨股
   - 区间成交：找大容量股
   - 区间净额：找主力票
   - 板块强度：找强势板块
4. 判断资金形态：
   - 持续净流入：主线热点
   - 首次净流入：启动信号
   - 净流出：行情结束
```

**资金形态判断**：

| 形态 | 信号 | 操作 |
|------|------|------|
| 持续净流入 | 主线热点 | 可追 |
| 首次净流入 | 启动初期 | 可入 |
| 净流出 | 行情结束 | 退出 |
| 反复流入 | 震荡行情 | 低吸 |

## 二、计划阶段（T日 18:00-21:00）

### 2.1 盘前计划制定

**核心任务**：制定T+1日的交易计划

**计划内容**：

#### 仓位计划

根据情绪周期决定仓位：

| 周期 | 建议仓位 |
|------|----------|
| 启动期 | 5-7成 |
| 发酵期 | 7-9成 |
| 高潮期 | 8-10成 |
| 分歧期 | 3-5成 |
| 退潮期 | 0-2成 |
| 冰点期 | 0成 |

#### 标的计划

从复盘选出的股票中确定：

| 优先级 | 股票 | 买入条件 | 仓位 | 止损价 |
|--------|------|----------|------|--------|
| 1 | | | | |
| 2 | | | | |
| 3 | | | | |

#### 买入条件（明确可执行）

```
□ 竞价高开 ___% 以上
□ 竞价量能 ___ 万手以上
□ 大盘环境：___（好/一般/差）
□ 板块配合：有板块效应
□ 题材配合：有新题材/政策
```

#### 卖出计划

```
止损规则（必须执行）：
- 单笔亏损 ___% 必须止损
- 收盘跌破 ___ 日线必须走
- 大盘急跌 ___% 必须走

止盈规则：
- 盈利 ___% 考虑分批卖
- 出现滞涨信号卖出
- 跌破 ___ 日线减半仓
```

## 三、执行阶段（T+1日）

### 3.1 盘前阶段（8:30-9:25）

**核心任务**：确认今日方向，准备开盘

#### 8:30-8:45 隔夜单处理

**使用功能**：隔夜委托 / 自选股监控

**操作**：

```
1. 查看隔夜单变化
2. 9:15-9:20 观察撤单情况
3. 9:20-9:25 决定是否撤单
```

**判断标准**：

| 信号 | 含义 | 操作 |
|------|------|------|
| 抢筹 | 隔夜单大增 | 排队靠前 |
| 抢跑 | 隔夜单大减 | 及时撤单 |

#### 8:45-9:00 消息解读

**使用功能**：开盘啦资讯 / 财经新闻

**操作**：

```
1. 查看隔夜重要消息
2. 解读对持仓/计划股的影响
3. 决定是否调整计划
```

#### 9:00-9:15 竞价观察

**使用功能**：集合竞价（开盘啦核心功能）

**操作**：

```
1. 首页 → 点击「集合竞价」
2. 观察计划股的竞价情况
3. 记录竞价涨幅、量能
```

**竞价形态判断**（来自官网）：

| 竞价形态 | 信号 | 操作 |
|----------|------|------|
| 高开5%+ | 抢筹 | 符合题材可追 |
| 高开2-5% | 强势 | 等待回调买 |
| 平开 | 观望 | 等盘中信号 |
| 低开2-5% | 弱势 | 观望为主 |
| 低开5%+ | 抢跑 | 放弃 |

#### 9:15-9:25 最终确认

**操作流程**：

```
□ 今日是否开仓？
   ├── 昨日涨停表现好 → 可开仓
   ├── 昨日跌停多 → 不开仓
   ├── 竞价抢筹多 → 积极开仓
   └── 竞价抢跑多 → 谨慎开仓

□ 计划股是否按预期开？
   ├── 符合 → 按计划买
   ├── 超预期 → 临时加仓
   └── 不及预期 → 放弃/观察
```

---

### 3.2 盘中阶段（9:30-14:30）

**核心任务**：执行计划，把握机会

#### 9:30-10:00 开盘博弈

**使用功能**：异动监控、涨跌停监控

**关注重点**：

| 信号 | 含义 | 操作 |
|------|------|------|
| 龙头股涨停 | 强化板块 | 买跟风 |
| 龙头股炸板 | 板块风险 | 放弃/跑路 |
| 板块爆发 | 主线确定 | 追最强 |

#### 10:00-11:30 上午盘

**使用功能**：板块联动、资金流向

**操作**：

```
1. 观察板块联动
   - 龙头涨停后，跟风是否启动
   - 板块是否有扩散

2. 观察资金流向
   - 资金是否在持续流入
   - 是否在切换板块

3. 执行买入
   - 符合计划条件 → 买入
   - 不符合 → 放弃
```

#### 11:30-13:00 午间复盘

**使用功能**：上午盘总结

**操作**：

```
1. 记录上午操作
2. 下午计划股是否有变化
3. 是否需要调整下午策略
```

#### 13:00-14:30 下午盘

**使用功能**：异动监控

**原则**：

| 时段 | 策略 |
|------|------|
| 13:00-14:00 | 观察为主 |
| 14:00-14:30 | 尾盘不买 |

**盘中应变**：

```
发现信号 → 快速判断 → 符合计划？→ 是→ 执行买入
                                      ↓否
                                放弃/观察
```

---

### 3.3 盘后阶段（14:30-15:00）

**核心任务**：记录交易，准备复盘

**使用功能**：交易记录

**记录内容**：

```
1. 今日交易记录
   - 买入：股票、价格、数量、理由
   - 卖出：股票、价格、数量、理由

2. 计划执行情况
   - 计划执行了___%
   - 临时调整原因：___

3. 明日提示
   - 持仓股处理：___
   - 自选股跟踪：___
   - 备选标的：___
```

## 五、实战案例

### 案例：主升浪行情（2025年12月）

**背景**：上证12/17-12/24六连阳，主升行情

#### 复盘阶段

```
1. 情绪判断
   - 涨停家数：持续增加 → 发酵期
   - 最高板：5板 → 强势
   - 建议仓位：7-9成

2. 涨停板复盘
   - 商业航天板块批量涨停
   - 区间涨幅前排：商业航天
   - 选出：龙头+跟风组合

3. 资金流向
   - 区间净流入：锂电池、通信超百亿
   - 细分：电解液、锂矿
   - 主力方向明确

4. 结论
   - 主线：商业航天
   - 支线：锂电池
   - 明日计划：买商业航天龙头
```

#### 计划阶段

```
1. 仓位：8成
2. 标的：
   - 首选：商业航天龙头
   - 次选：跟风最强
   - 备选：锂电池
3. 买入条件：
   - 竞价高开2%+
   - 板块有配合
   - 大盘不跳水
4. 止损：-5%
```

#### 执行阶段

```
1. 盘前
   - 商业航天龙头高开3%，符合条件
   - 板块整体高开，有配合

2. 盘中
   - 9:45 龙头涨停，排队买入跟风
   - 10:30 跟风涨停，持股待涨
   - 14:00 板块开始回落，减半仓

3. 盘后
   - 记录操作
   - 准备明日复盘
```

## 六、关键提醒

### 6.1 开盘啦核心原则

1. **永远跟资金走** - 资金流入才是真的
2. **永远跟龙头走** - 龙头是市场最强
3. **永远跟情绪走** - 情绪决定仓位
4. **永远守纪律** - 止损是底线

### 6.2 常见错误

| 错误 | 后果 | 纠正 |
|------|------|------|
| 不看情绪乱买 | 追高被套 | 先判断周期 |
| 不设止损 | 大亏 | -5%必走 |
| 买跟风 | 涨幅小 | 只买龙头 |
| 随意改计划 | 乱交易 | 按计划执行 |

### 6.3 每日检查清单

```
□ 情绪周期判断
□ 涨停板复盘完成
□ 炸板分析完成
```
