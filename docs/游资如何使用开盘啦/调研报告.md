# 游资如何使用开盘啦进行复盘、计划与执行

> 本文档系统梳理游资（短线活跃资金）如何利用开盘啦进行每日交易闭环：复盘→计划→执行计划
> 涵盖每个阶段的核心任务、使用功能、判断标准和操作流程

---

## 一、开盘啦核心功能全景图

> 根据开盘啦官网介绍：「开盘啦是市场上最专业和及时的资金数据分析工具，拥有集合竞价异动、狙击龙头等多个强力功能，增加选股成功率。」

### 1.1 主要功能模块

| 模块 | 核心价值 | 游资使用频率 |
|------|----------|--------------|
| **涨停板分析** | 捕捉市场最强信号 | ★★★★★ |
| **炸板分析** | 识别失败涨停、风险预警 | ★★★★☆ |
| **龙虎榜** | 追踪主力资金动向 | ★★★★★ |
| **资金流向** | 判断主力进出 | ★★★★☆ |
| **涨停原因** | 理解上涨逻辑 | ★★★★☆ |
| **板块轮动** | 把握热点切换 | ★★★★☆ |
| **情绪指标** | 判断市场周期位置 | ★★★★★ |
| **异动监控** | 实时捕捉机会 | ★★★☆☆ |
| **区间统计** | 阶段性资金分析 | ★★★★☆ |
| **集合竞价** | 盘前异动捕捉 | ★★★★★ |

### 1.2 开盘啦核心特色（来自官网）

根据开盘啦官方介绍，其核心功能包括：

1. **实时龙虎榜盯盘** - 掌握资金动向
   - 区间净额功能叠加时间轴，快速定位大盘不同节点资金净流入情况
   - 指数调整时逆势上攻的个股，说明大资金主动大幅买入

2. **玩转题材和资金流** - 捕捉主升行情机会
   - 通过板块区间强度、区间净额、区块间成交等维度找到资金进攻方向
   - 从资金角度挖掘净流入超百亿的板块（如锂电池、通信）
   - 区间统计从全市场5000+个股中挖掘独立个股，自下而上寻找市场脉络

3. **情绪大师** - 精准拿捏市场节奏
   - 上涨家数、量能萎缩度判断市场情绪
   - 盘中量能变化判断昨日强势题材能否加强
   - 权重表现模块定位对指数贡献度强弱的板块
   - 涨停梯队识别（一进二、二进三、三板以上）判断主升期特征

### 1.2 游资核心使用场景

```
T日收盘后 ──→ 复盘阶段 ──→ 创建T+1日计划
     │
     ├── 涨停板复盘 → 识别龙头/跟风
     ├── 炸板复盘 → 总结失败原因
     ├── 龙虎榜复盘 → 追踪主力资金
     ├── 资金流向复盘 → 判断板块趋势
     ├── 情绪复盘 → 确定周期位置
     │
T+1日开盘前 ──→ 计划执行 ──→ 盘前/盘中/盘后
     │
     ├── 盘前：确认方向、选股池
     ├── 盘中：跟随信号、执行买卖
     └── 盘后：记录交易、准备复盘
```

## 二、复盘阶段（T日 15:00-18:00）

### 2.1 涨停板复盘

**目标**：识别今日最强股票，找到明日潜在标的

**使用功能**：
1. **涨停板池** - 查看当日所有涨停股
   - 按涨停时间排序：越早涨停越强
   - 按封单金额排序：封单越大越强
   - 按流通市值排序：小市值更容易炒

2. **连板池** - 识别当前市场龙头
   - 3板以上：妖股候选
   - 5板以上：市场核心标的
   - 断板股：可能反包

3. **涨停时间分布** - 判断市场情绪
   - 上午涨停家数多：强势市场
   - 下午涨停家数多：轮动补涨
   - 尾盘涨停家数多：情绪末端

**判断标准**：

| 指标 | 强势信号 | 弱势信号 |
|------|----------|----------|
| 涨停家数 | >50家 | <20家 |
| 最高板 | 5板以上 | 2-3板 |
| 涨停封板率 | >90% | <70% |
| 炸板率 | <15% | >30% |
| 跌停家数 | 0-3家 | >10家 |

**输出**：
- 记录当日最高板、龙头股
- 标记最强板块
- 选出3-5只明日重点关注股

### 2.2 炸板分析

**目标**：总结失败原因，避免重复犯错

**使用功能**：
1. **炸板池** - 当日被炸股票列表
   - 炸板时间：越早炸越弱
   - 炸板次数：多次炸板极弱
   - 回落幅度：回落越大越弱

2. **炸板原因分类**：
   - 跟风股炸板：龙头已封住，跟风被炸（弱）
   - 板块炸板：整个板块崩盘（风险）
   - 量能不足：封板资金不够（技术面）
   - 大盘拖累：指数大跌拖累（系统性）

**判断标准**：

| 炸板类型 | 风险等级 | 处理建议 |
|----------|----------|----------|
| 跟风炸板 | 中 | 放弃关注 |
| 龙头炸板 | 低 | 可能反包 |
| 量能不足 | 中 | 观察量价 |
| 大盘拖累 | 高 | 谨慎出手 |

**输出**：
- 记录炸板股及原因
- 反思买入逻辑是否正确
- 总结操作失误

### 2.3 龙虎榜复盘

**目标**：追踪主力资金动向，识别合力票

**使用功能**：
1. **龙虎榜列表** - 当日上榜股票
   - 机构买入：长期资金
   - 游资买入：短线资金
   - 合力买入：最佳标的

2. **游资动向**：
   - 知名游资席位
   - 买入金额排行
   - 卖出金额排行

3. **筹码分析**：
   - 机构买入占比
   - 游资买入占比
   - 净买入金额

**判断标准**：

| 榜单类型 | 信号解读 |
|----------|----------|
| 机构大幅买入 | 看好基本面，可能走趋势 |
| 游资合力买入 | 短线强势，可能继续涨 |
| 游资对倒 | 可能出货，谨慎 |
| 机构游资分歧 | 观望为主 |

**输出**：
- 记录知名游资买入标的
- 识别机构+游资合力票
- 排除主力对倒出货股

### 2.4 资金流向复盘

**目标**：判断板块趋势，把握主线热点

**使用功能**：
1. **主力净流入** - 按板块/个股排序
   - 板块净流入：热点方向
   - 个股净流入：具体标的

2. **资金趋势**：
   - 5日主力净流入
   - 10日主力净流入
   - 主力建仓/出货迹象

3. **板块轮动**：
   - 今日热点板块
   - 昨日热点板块
   - 板块持续性判断

**判断标准**：

| 资金形态 | 信号含义 |
|----------|----------|
| 板块持续净流入 | 主线热点，可追 |
| 板块首次净流入 | 启动初期，可入 |
| 板块净流出 | 行情结束，退出 |
| 板块间轮动 | 震荡市，低吸为主 |

**输出**：
- 确定当日主线板块
- 识别资金切换方向
- 选出板块内最强股

### 2.5 情绪周期复盘

**目标**：确定当前市场周期位置，决定仓位和策略

> 根据开盘啦「情绪大师」功能的官方介绍：
> - 当上涨家数＜1000家，量能萎缩，说明市场情绪消极，若量能放大则说明情绪恐慌，有资金出逃
> - 涨停梯队（一进二、二进三、三板以上）说明资金承接有序，是主升期的特征；梯队断层则预示情绪走弱
> - 涨停破板率高，打板容错率低；涨停破板率低，资金做多共识强
> - 昨日涨停今表高，说明昨日打板溢价高、赚钱效应好
> - 昨日连扳今表高，说明连扳风格赚钱效应好；反之未晋级个股可能存在较大负反馈

**使用功能**：
1. **情绪指标**：
   - 涨停家数
   - 炸板率
   - 跌停家数
   - 昨日连板溢价
   - 昨日涨停表现

2. **周期阶段识别**：
   - 启动期 → 发酵期 → 高潮期 → 分歧期 → 退潮期 → 冰点期

3. **龙头股表现**：
   - 最高板涨幅
   - 龙头股反馈
   - 补涨股表现

4. **量能分析**（开盘啦特色）：
   - 市场量能功能可以盘中实时预测当天沪深两市总成交额
   - 盘中量能变化可以判断昨日强势题材能否加强
   - 盘中放量节点配合板块区间强度找资金进攻方向
   - 历史量能配合上证大盘指数日K，日线级别判断趋势

**判断标准**：

| 周期阶段 | 特征 | 策略 |
|----------|------|------|
| 启动期 | 涨停家数增加，最高板出现 | 试错龙头 |
| 发酵期 | 板块效应出现，龙头继续涨 | 积极参与 |
| 高潮期 | 涨停家数峰值，板块全面爆发 | 持股待涨 |
| 分歧期 | 龙头炸板，板块分化 | 谨慎追高 |
| 退潮期 | 跌停增加，龙头A杀 | 空仓休息 |
| 冰点期 | 涨停家数极低，市场绝望 | 等待新周期 |

**输出**：
- 确定当前周期阶段
- 决定明日仓位策略
- 调整选股标准

## 三、计划阶段（T日 18:00-21:00）

### 3.1 盘前计划制定

**核心任务**：制定T+1日的交易计划

**计划内容**：

1. **仓位计划**
   - 根据情绪周期决定仓位
   - 退潮期/冰点期：0-3成仓
   - 启动期/发酵期：5-7成仓
   - 高潮期：8-10成仓

2. **标的计划**
   - 从复盘选出的3-5只股票中确定
   - 按优先级排序：第一标的、第二标的...
   - 设定每只股票的买入条件

3. **买入计划**
   - 买入价格区间
   - 买入仓位分配
   - 加仓/补仓条件

4. **卖出计划**
   - 止损价：-5%必须走
   - 止盈价：+10%考虑分批卖
   - 盘中应变条件

### 3.2 盘前精选（次交易日 9:00-9:25）

**使用功能**：

1. **隔夜委托**：
   - 昨日涨停股表现
   - 隔夜单排队情况
   - 撤单/改单

2. **竞价观察**：
   - 昨日计划股竞价情况
   - 竞价抢筹/抢跑信号
   - 开盘预期

3. **消息面**：
   - 隔夜消息解读
   - 个股公告
   - 政策/行业新闻

**判断标准**：

| 竞价形态 | 信号 | 操作 |
|----------|------|------|
| 高开高走 | 强势 | 符合预期可买 |
| 高开低走 | 不及预期 | 放弃或观察 |
| 低开高走 | 弱转强 | 可考虑买 |
| 低开低走 | 弱势 | 放弃 |

### 3.3 计划执行检查清单

```
□ 情绪周期判断正确
□ 仓位符合周期特征
□ 标的符合主线逻辑
□ 买入条件明确
□ 止损线设定
□ 止盈计划制定
□ 备用方案准备
```

## 四、执行阶段（T+1日 9:30-15:00）

### 4.1 盘前（9:00-9:25）

**任务**：确认今日方向，准备开盘

**操作流程**：

1. **隔夜单处理**
   - 9:15-9:20：观察隔夜单变化
   - 9:20-9:25：决定是否撤单
   - 抢筹股：排队靠前
   - 抢跑股：及时撤单

2. **开盘确认**
   - 竞价涨停股：强势信号
   - 竞价跌停股：风险信号
   - 板块联动：确认主线

**决策点**：
- 昨日计划股是否按预期开
- 是否需要临时调整计划
- 是否出现新的机会

### 4.2 盘中（9:30-11:30 / 13:00-14:30）

**任务**：执行计划，随机应变

**使用功能**：

1. **异动监控**：
   - 快速拉升股
   - 直线涨停股
   - 大单买入股

2. **板块联动**：
   - 板块内个股联动
   - 龙头股走势
   - 跟风股表现

3. **分时量价**：
   - 量价配合
   - 量价背离
   - 突破/回落信号

**执行原则**：

| 情况 | 正确操作 |
|------|----------|
| 符合预期 | 按计划买入 |
| 超预期 | 临时加仓 |
| 不及预期 | 放弃或观察 |
| 出现风险 | 立即止损 |

**盘中决策流程**：

```
发现信号 → 快速判断 → 符合计划？→ 是→ 执行买入
                                      ↓否
                                放弃/观察
```

### 4.3 盘后（14:30-15:00）

**任务**：记录交易，准备复盘

**记录内容**：

1. **交易记录**
   - 买入股票、价格、数量
   - 卖出股票、价格、数量
   - 买卖理由
   - 结果总结

2. **计划对比**
   - 计划执行情况
   - 临时调整原因
   - 经验教训

3. **明日提示**
   - 持仓股处理
   - 自选股跟踪
   - 备选标的

## 五、三阶段实战流程

### 5.1 盘前阶段（每日 8:30-9:25）

**核心任务**：确定今日方向，做好交易准备

| 时间段 | 任务 | 使用功能 | 判断标准 |
|--------|------|----------|----------|
| 8:30-8:45 | 隔夜单处理 | 隔夜委托 | 抢筹or抢跑 |
| 8:45-9:00 | 消息解读 | 财经要闻 | 利好/利空 |
| 9:00-9:15 | 竞价观察 | 竞价监控 | 强弱判断 |
| 9:15-9:25 | 最终确认 | 竞价排名 | 买入/放弃 |

**关键决策**：

```
今日是否开仓？
├── 昨日涨停表现好 → 可开仓
├── 昨日跌停多 → 不开仓
├── 竞价抢筹多 → 积极开仓
└── 竞价抢跑多 → 谨慎开仓
```

### 5.2 盘中阶段（每日 9:30-14:30）

**核心任务**：执行计划，把握机会

| 时段 | 任务 | 重点关注 | 原则 |
|------|------|----------|------|
| 9:30-10:00 | 开盘博弈 | 龙头股走势 | 强者恒强 |
| 10:00-11:00 | 上午盘 | 板块发酵 | 顺势而为 |
| 11:00-11:30 | 上午收官 | 尾盘异动 | 谨慎追高 |
| 13:00-14:00 | 下午盘 | 资金回流 | 观察为主 |
| 14:00-14:30 | 尾盘 | 抢筹股 | 尾盘不买 |

**实战技巧**：

1. **龙头股买入时机**：
   - 首次分歧后低吸
   - 板上确认
   - 回调到5日线

2. **跟风股买入时机**：
   - 龙头涨停后，跟风启动时
   - 板块首次爆发时
   - 不可追高

3. **止损纪律**：
   - -5%必须走
   - 破位必须走
   - 大盘急跌必须走

### 5.3 盘后阶段（每日 15:00-18:00）

**核心任务**：完成复盘，制定明日计划

| 时间 | 任务 | 输出 |
|------|------|------|
| 15:00-15:30 | 记录交易 | 交易清单 |
| 15:30-16:30 | 涨停板复盘 | 龙头股池 |
| 16:30-17:30 | 炸板/龙虎榜 | 风险/机会 |
| 17:30-18:00 | 情绪周期 | 周期判断 |

## 六、游资实战案例

### 6.1 龙头股战法

**场景**：市场处于发酵期，出现3板龙头

**复盘阶段**：
1. 涨停板分析：确认3板龙头
2. 板块分析：确认有板块效应
3. 资金分析：主力净流入
4. 选出：龙头+跟风组合

**计划阶段**：
1. 明日计划买入龙头
2. 买入条件：竞价高开3%+ 或 盘中首次分歧
3. 仓位：5成
4. 止损：-5%

**执行阶段**：
1. 竞价高开4%，符合条件，直接买
2. 涨停后持股
3. 尾盘不板卖出半仓

### 6.2 空仓等待战法

**场景**：市场处于退潮期，龙头A杀

**复盘阶段**：
1. 涨停家数：10家（少）
2. 最高板：2板（低）
3. 炸板率：40%（高）
4. 跌停家数：20家（多）

**判断**：退潮期，空仓为主

**计划阶段**：
1. 明日不开新仓
2. 只处理持仓
3. 等待冰点机会

**执行阶段**：
1. 持仓股-5%止损
2. 不开新仓
3. 观望等待

## 九、总结：游资每日工作流

```
┌─────────────────────────────────────────────────────────────────┐
│                      T日（复盘+计划）                            │
├─────────────────────────────────────────────────────────────────┤
│  15:00-16:00  │  记录今日交易                                  │
│  16:00-17:00  │  涨停板复盘 → 找龙头                          │
│  17:00-17:30  │  炸板复盘 → 总结教训                          │
│  17:30-18:00  │  龙虎榜+资金流向 → 跟主力                     │
│  18:00-19:00  │  情绪周期判断 → 定仓位                        │
│  19:00-20:00  │  制定明日计划 → 选标的                         │
│  20:00-21:00  │  设置隔夜委托（可选）                          │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    T+1日（执行+复盘）                           │
├─────────────────────────────────────────────────────────────────┤
│  8:30-9:15    │  隔夜单处理+消息解读                           │
│  9:15-9:25    │  竞价观察 → 最终确认                           │
│  9:30-11:30   │  盘中执行 → 买入/卖出/持仓                    │
│  11:30-13:00  │  午间复盘 → 下午计划                           │
│  13:00-14:30  │  下午执行 → 盘中应变                           │
│  14:30-15:00  │  盘后记录 → 交易总结                          │
└─────────────────────────────────────────────────────────────────┘
```

**核心原则**：
1. **计划你的交易，交易你的计划**
2. **永远跟随市场，不主观意淫**
3. **止损是底线，纪律大于收益**
4. **情绪周期决定仓位**
5. **龙头优先，跟风次之**
