### 2.3 核心指标解读

| 指标 | 说明 | 强势信号 |
|------|------|----------|
| 涨停时间 | 首次涨停的时间 | 越早越强 |
| 封单金额 | 涨停板上的买单金额 | 越大越强 |
| 流通市值 | 股票流通市值 | 越小越容易炒 |
| 换手率 | 当日换手比例 | 适中最好 |
| 涨停原因 | 上涨驱动因素 | 题材越新越好 |

### 2.4 筛选功能

1. **按连板数筛选**
   - 首板：新启动的股票
   - 2-3连板：确认强势
   - 4-5连板：龙头候选
   - 6板+：妖股

2. **按时间筛选**
   - 开盘集合竞价（09:25）
   - 上午（09:30-11:30）
   - 下午（13:00-15:00）

3. **按涨停原因筛选**
   - 新题材
   - 业绩预增
   - 并购重组
   - 热点概念

## 十二、常见问题

### Q1：涨停板太多看不过来？
**A**：使用筛选功能，先按连板数筛选缩小范围，再按涨停时间排序。

### Q2：如何判断炸板股能不能买？
**A**：查看炸板原因，如果是跟风炸板则放弃，如果是龙头炸板可等待反包机会。

### Q3：龙虎榜数据怎么看？
**A**：重点关注「机构买入」和「知名游资买入」，避免「对倒」个股。

### Q4：资金流入可信吗？
**A**：结合5日/10日趋势看，持续流入比单日流入更可靠。

### Q5：情绪指标在哪看？
**A**：首页顶部直接显示，或者点击「情绪指标」进入详情页。

## 十三、区间统计功能（进阶）

> 来自开盘啦官方教程：「玩转题材和资金流，捕捉主升行情机会」

### 13.1 功能价值

区间统计是开盘啦的核心功能之一，能帮助游资：
- 从全市场5000+个股中挖掘独立个股
- 自下而上寻找市场脉络和机会
- 判断板块持续性
- 找到资金主攻的核心方向

### 13.2 使用方法

**路径**：资金流向 → 区间统计

**核心维度**：

| 维度 | 说明 | 用途 |
|------|------|------|
| 区间涨幅 | 特定时间段内的涨幅 | 找领涨股 |
| 区间成交 | 特定时间段内的成交量 | 找大容量股 |
| 区间净额 | 特定时间段内的资金净流入 | 找主力票 |
| 区间强度 | 相对市场的强弱程度 | 找强势股 |

### 13.3 实战案例

以2025年12月17日起上证主升行情为例：

1. **判断大盘环境**
   - 上证从3815.84低点逐步反弹
   - 12月24日实现六连阳
   - 沪深两市量能温和放大
   - 市场环境良好

2. **找主线板块**
   - 点击板块 → 商业航天板块趋势较好
   - 区间板块领涨个股均是主力资金大幅净流入
   - 判断：商业航天是和大盘共振的主线题材

3. **找目标个股**
   - 偏好强势股：按区间涨幅排行找领涨股
   - 偏好大容量：选择区间大成交量个股
   - 后市商业航天走出强势主升浪

4. **从资金角度验证**
   - 区间资金净流入超百亿的板块：锂电池、通信
   - 进一步看细分：电解液、锂矿等
   - 点击个股区间净额直接找到资金净流入个股

## 十四、集合竞价功能

> 来自开盘啦官方介绍：「集合竞价异动」是开盘啦的核心功能之一

### 14.1 功能价值

集合竞价是盘前最重要的分析时段：
- 9:15-9:20：可以挂单也可以撤单
- 9:20-9:25：只能挂单不能撤单
- 9:25-9:30：接受挂单但不处理

### 14.2 开盘啦竞价监控

**路径**：首页 → 集合竞价

**监控内容**：

| 监控项 | 信号含义 |
|--------|----------|
| 竞价抢筹 | 高开5%+，资金抢入 |
| 竞价抢跑 | 低开5%+，资金抢出 |
| 竞价量能 | 竞价成交量异常放大 |
| 竞价异动 | 竞价期间大幅波动 |

### 14.3 竞价操作策略

| 竞价形态 | 信号 | 操作建议 |
|----------|------|----------|
| 高开5%+ | 抢筹 | 符合题材可追 |
| 高开2-5% | 强势 | 等待回调买 |
| 平开 | 观望 | 等盘中信号 |
| 低开2-5% | 弱势 | 观望为主 |
| 低开5%+ | 抢跑 | 放弃或做空 |

### 14.4 竞价看盘要点

1. **看板块**：竞价期间板块整体表现
2. **看龙头**：龙头股竞价情况
3. **看跟风**：跟风股是否被带动
4. **看量能**：竞价量能是否放大
5. **看消息**：是否有利好/利空
