# 第二部分：复盘阶段（做什么+为什么+怎么做）

## 2.1 情绪判断

### 做什么（任务）

1. 查看今日涨停家数、炸板率、跌停家数
2. 查看最高板连板数
3. 查看昨日涨停表现、昨日连板表现
4. 综合判断当前处于哪个周期阶段

### 为什么做（逻辑）

> **核心原理**：情绪是市场的「温度计」，决定赚钱难度
>
> - 牛市（高潮期）：80%的股票都在涨，你随便买都可能赚
> - 熊市（退潮期）：80%的股票都在跌，你买什么都可能亏
>
> 情绪周期本质是「供需关系」的周期性表现：
> - 买的人多（情绪高）→ 股票供不应求 → 价格上涨
> - 卖的人多（情绪低）→ 股票供过于求 → 价格下跌

**为什么要先判断情绪？**

```
情绪 → 决定赚钱难度 → 决定仓位 → 决定策略
```

| 周期 | 赚钱难度 | 正确策略 |
|------|----------|----------|
| 启动期 | 中 | 试错龙头 |
| 发酵期 | 易 | 积极参与 |
| 高潮期 | 易 | 持股待涨 |
| 分歧期 | 难 | 谨慎追高 |
| 退潮期 | 极难 | 空仓休息 |
| 冰点期 | 极难 | 等待 |

### 怎么做（操作）

**使用功能**：情绪指标 / 情绪大师

**操作步骤**：

```
步骤1：打开开盘啦 → 首页顶部查看情绪仪表盘

步骤2：记录关键数据
┌─────────────────────────────────────────┐
│ 指标            │  今日数据  │  强弱判断  │
├─────────────────────────────────────────┤
│ 涨停家数        │  ___家    │  强>50/中20-50/弱<20  │
│ 最高板          │  ___板    │  强5+/中3-4/弱1-2     │
│ 炸板率          │  ___%    │  低<15/中15-30/高>30   │
│ 跌停家数        │  ___家    │  低0-3/中4-10/高>10    │
│ 昨日涨停溢价    │  ___%    │  高>3/中0-3/低<0       │
└─────────────────────────────────────────┘

步骤3：判断周期阶段
- 涨停50+、最高5板+、炸板<15%、跌停<3家 → 高潮期
- 涨停20-50、最高3-4板、炸板15-30%、跌停3-10家 → 发酵期/分歧期
- 涨停<20、最高<3板、炸板>30%、跌停>10家 → 退潮期/冰点期

步骤4：根据周期决定仓位
- 启动/发酵/高潮：5-10成仓
- 分歧：3-5成仓
- 退潮/冰点：0-2成仓
```

## 2.2 涨停板复盘

### 做什么（任务）

1. 查看当日所有涨停股
2. 筛选连板股，识别龙头
3. 记录涨停时间、封单金额、流通市值、涨停原因
4. 选出3-5只明日重点关注股

### 为什么做（逻辑）

> **核心原理**：涨停板是市场最强信号
>
> - 涨停 = 资金用真金白银投票「这个股票值得买」
> - 涨停板数量多 = 大量资金在积极做多 = 强势市场
> - 涨停板数量少 = 资金入场意愿低 = 弱势市场

**为什么要按涨停时间排序？**

```
越早涨停 = 资金越看好
09:25涨停：开盘就涨停，资金一开盘就迫不及待要买
14:30涨停：到尾盘才涨停，资金是没办法才拉的
```

**为什么要区分龙头和跟风？**

```
龙头涨停 → 带动板块 → 跟风跟着涨
跟风涨停 → 只是跟随 → 没有带动性

龙头的溢价来自「领涨」（主动）
跟风的溢价来自「被带动」（被动）

龙头70%继续涨，跟风只有30%
```

### 怎么做（操作）

**使用功能**：涨停板分析、连板池

**操作步骤**：

```
步骤1：首页 → 点击「涨停板」

步骤2：查看全部涨停股
- 按「涨停时间」排序：越早越强
- 按「封单金额」排序：越大越强
- 按「流通市值」排序：越小越容易炒

步骤3：记录涨停股信息
┌─────────────────────────────────────────────────────────────────────┐
│ 时间    │ 代码    │ 名称      │ 几连板 │ 涨停原因  │  备注       │
├─────────────────────────────────────────────────────────────────────┤
│ 09:25  │ 600XXX │ 贵州茅台  │ 3板    │ 业绩预增  │  龙头候选   │
│ 09:30  │ 000XXX │ 五粮液    │ 2板    │ 白酒概念  │  跟风       │
│ 10:15  │ 300XXX │ 某科技    │ 首板    │ AI概念   │  新题材     │
└─────────────────────────────────────────────────────────────────────┘

步骤4：筛选连板股 → 识别龙头
- 5板以上：市场核心（重点关注）
- 3-4板：跟风龙头（次要关注）
- 2板以下：首板（观察为主）

步骤5：筛选「首板」找新题材
- 找新题材（首次出现的概念）
- 找小市值（<50亿更容易炒）
- 找涨停时间早的

步骤6：选出3-5只明日重点关注
┌────────────────────────────────────────────────────────┐
│ 优先级  │  股票代码  │  股票名称  │  关注理由       │
├────────────────────────────────────────────────────────┤
│ 1       │  600XXX   │  贵州茅台  │  3板龙头/业绩预增 │
│ 2       │  300XXX   │  某科技    │  首板/新题材/小市值│
│ 3       │  000XXX   │  五粮液    │  2板/跟风龙      │
└────────────────────────────────────────────────────────┘
```

## 2.3 炸板复盘

### 做什么（任务）

1. 查看当日所有炸板股
2. 分析炸板原因（跟风/板块/量能/大盘）
3. 反思自己的操作是否涉及类似问题

### 为什么做（逻辑）

> **核心原理**：炸板 = 资金做多失败
>
> - 本来看好要涨停 → 结果没封住 → 资金跑了
> - 炸板后第二天低开是大概率事件

**为什么要分析炸板原因？**

```
不同原因，后续走势完全不同：

跟风炸板：资金不认可这个股票 → 大概率继续跌
龙头炸板：获利盘卖出 → 可能反包
板块炸板：整个板块不行 → 规避整个板块
量能不足：资金不够 → 观察
```

**跟风炸板为什么不能买？**

```
1. 龙头涨停 → 跟风涨停
2. 龙头没封住 → 跟风先炸
3. 跟风炸了说明：资金不认可这个股票
4. 第二天低开是大概率事件（70%以上）
```

### 怎么做（操作）

**使用功能**：炸板分析

**操作步骤**：

```
步骤1：首页 → 点击「炸板分析」

步骤2：查看炸板股列表
- 炸板时间：越早炸越弱
- 炸板次数：多次炸极弱
- 回落幅度：越大越弱

步骤3：分析炸板原因
┌─────────────────────────────────────────────────────────────────┐
│ 炸板类型  │  特征                           │  风险等级  │
├─────────────────────────────────────────────────────────────────┤
│ 跟风炸板  │  龙头已封住，跟风被炸           │  高        │
│ 板块炸板  │  整个板块回落                   │  高        │
│ 量能不足  │  封板资金不够                   │  中        │
│ 大盘拖累  │  大盘大跌拖累                   │  中        │
└─────────────────────────────────────────────────────────────────┘

步骤4：记录反思
┌────────────────────────────────────────────────────────┐
│ 炸板股票  │  炸板原因  │  反思                    │
├────────────────────────────────────────────────────────┤
│  XXX     │  跟风炸板  │  不该买跟风股             │
│  XXX     │  板块炸板  │  板块已经高潮该跑        │
└────────────────────────────────────────────────────────┘

步骤5：总结教训
- 今天打板为什么炸？
- 是选的股不行，还是买的时机不对？
- 有什么经验教训？
```

## 2.4 龙虎榜复盘

### 做什么（任务）

1. 查看今日机构买入、游资买入的股票
2. 识别知名游资席位动向
3. 找出机构+游资合力买入的股票
4. 排除对倒出货的股票

### 为什么做（逻辑）

> **核心原理**：龙虎榜 = 主力资金动向的「监控摄像头」
>
> - 散户买卖：几百几千股，对股价影响微乎其微
> - 主力买卖：几万几十万股，直接决定股价涨跌
> - 跟踪主力 = 站在资金这一边

**为什么要区分机构和游资？**

```
不同资金，操作风格完全不同：

机构资金：
- 操作风格：价值投资
- 持有时间：几周-几个月
- 拉升方式：缓慢推升

游资资金：
- 操作风格：趋势投机
- 持有时间：几天-几周
- 拉升方式：快速拉升
```

**为什么机构+游资合力最好？**

```
机构买入 = 基本面认可，长期看好
游资买入 = 技术面认可，短期要涨
两者合力 = 既要涨，又要涨得久
```

**为什么要避免「对倒」？**

```
对倒 = 左手卖右手，制造虚假成交量

目的：吸引散户跟进，然后出货
结果：散户追高被套

特征：
- 买入 ≈ 卖出
- 今天买，明天卖
- 营业部反复出现
```

### 怎么做（操作）

**使用功能**：龙虎榜

**操作步骤**：

```
步骤1：首页 → 点击「龙虎榜」

步骤2：筛选「机构买入」
- 记录净买入前5
- 机构买入 = 看好基本面，可能走趋势

步骤3：筛选「游资买入」
- 记录知名游资席位动向
- 知名游资：华鑫证券上海宛平南路、东方财富拉萨团结路等

步骤4：分析合力
┌─────────────────────────────────────────────────────────────────┐
│ 股票代码  │  机构买入  │  游资买入  │  合力判断          │
├─────────────────────────────────────────────────────────────────┤
│ 600XXX   │  2亿       │  1.5亿     │  合力，可关注      │
│ 000XXX   │  5000万    │  1000万    │  游资主导          │
│ 300XXX   │  3000万    │  2800万    │  对倒，规避        │
└─────────────────────────────────────────────────────────────────┘

步骤5：排除问题股
- 对倒股（买入≈卖出）
- 温州帮等操纵股
- 高位放量出货股

步骤6：选出明日关注标的
- 机构+游资合力最佳
- 纯机构可中线
- 纯游资需快进快出
```

## 2.5 资金流向复盘

### 做什么（任务）

1. 查看今日主力净流入的板块
2. 使用区间统计查看5日/10日持续净流入的板块
3. 识别主线板块（资金持续流入+涨停多+涨幅大）
4. 从主线板块中选出最强个股

### 为什么做（逻辑）

> **核心原理**：资金是股价涨跌的根本
>
> - 股价上涨的唯一原因：钱多
> - 供需关系：买的人多（资金流入）→ 供不应求 → 价格上涨

**为什么要用区间统计？**

```
单日资金流入可能是假象：
今天资金流入 = 有可能明天就跑
连续5天资金流入 = 大概率是真的要涨

区间统计的意义：
单日净流入：脉冲行情，持续性不确定
5日净流入：资金持续关注，可信度高
10日净流入：资金深度布局，大行情
```

**为什么要找持续净流入？**

```
持续净流入 = 主力真金白银在建仓

逻辑链：
持续净流入 → 主力在吸筹 → 后面要拉升 → 现在是低位 → 应该买
```

### 怎么做（操作）

**使用功能**：资金流向、区间统计

**操作步骤**：

```
步骤1：首页 → 点击「资金流向」

步骤2：切换「板块」→ 查看主力净流入排行
- 记录净流入前5板块
- 今日净流入 > 10亿 = 强

步骤3：使用「区间统计」功能
- 资金流向 → 区间统计
- 选择「近5日」或「近10日」
- 筛选持续净流入的板块

步骤4：识别主线板块
┌─────────────────────────────────────────────────────────────────┐
│ 主线特征          │  判断标准                                 │
├─────────────────────────────────────────────────────────────────┤
│ 资金持续净流入    │  连续3天+净流入                          │
│ 板块效应强        │  涨停股数量 > 5只                         │
│ 与大盘共振        │  大盘涨，板块涨更多                      │
└─────────────────────────────────────────────────────────────────┘

步骤5：从主线板块选个股
- 区间涨幅排行：找领涨股
- 区间成交排行：找大容量股
- 区间净额排行：找主力票

步骤6：确认资金形态
┌─────────────────────────────────────────────────────────────────┐
│ 形态        │  信号              │  操作                      │
├─────────────────────────────────────────────────────────────────┤
│ 持续净流入  │  主线热点          │  可追                      │
│ 首次净流入  │  启动信号          │  可入                      │
│ 净流出      │  行情结束          │  退出                      │
│ 反复流入    │  震荡行情          │  低吸                      │
└─────────────────────────────────────────────────────────────────┘
```

# 第三部分：计划阶段（做什么+为什么+怎么做）

## 3.1 制定明日交易计划

### 做什么（任务）

1. 根据情绪判断确定明日仓位
2. 从复盘选出的股票中确定目标标的
3. 明确每只股票的买入条件
4. 设定止损线和止盈计划

### 为什么做（逻辑）

> **核心原理**：计划你的交易，交易你的计划
>
> - 没有计划 = 临盘冲动 = 大概率亏钱
> - 有计划 = 按计划执行 = 纪律保证

**为什么要明确买入条件？**

```
模糊的买入条件 = 没有条件
「涨幅大了就买」→ 涨多少算大？
「感觉好了就买」→ 什么时候算好？

明确的买入条件：
「竞价高开3%以上，且成交量大于昨日的1.5倍」
= 符合就买，不符合就不买
```

**为什么要设止损？**

```
数学原理：亏50%需要赚100%才能回本

亏5%：赚5.26%回本
亏10%：赚11.11%回本
亏20%：赚25%回本
亏50%：赚100%回本
亏70%：赚233%回本

小亏的目的：
- 保护本金
- 保持交易主动性
- 避免深套
```

### 怎么做（操作）

**操作步骤**：

```
步骤1：仓位计划

├────────────────────────────────────────────────────────┤

│ 发酵期    │  7-9成       │  积极参与                  │
│ 高潮期    │  8-10成      │  持股待涨                  │
│ 分歧期    │  3-5成       │  谨慎                      │
│ 退潮期    │  0-2成       │  空仓休息                  │
│ 冰点期    │  0成         │  等待                      │
└────────────────────────────────────────────────────────┘

步骤2：标的计划
从复盘选出的股票中确定：

┌─────────────────────────────────────────────────────────────────┐

# 第四部分：执行阶段（做什么+为什么+怎么做）

## 4.1 盘前阶段（8:30-9:25）

### 做什么（任务）

1. 处理隔夜单
2. 解读隔夜消息
3. 观察竞价情况
4. 最终确认是否开仓

### 为什么做（逻辑）

> **核心原理**：竞价 = 当天第一轮「资金投票」
>
> - 9:15-9:20：可以挂单也可以撤单（试探性）
> - 9:20-9:25：只能挂单不能撤单（真金白银）
> - 9:25：竞价结果确定

**竞价为什么重要？**

```
竞价是「盘前预演」：
开盘半小时的走势 = 竞价结果的延续

竞价高开高走 → 强势延续
竞价低开低走 → 弱势延续
```

**为什么要看竞价量能？**

```
量能 = 资金的参与程度

竞价量能大（比昨日放大）：
- 说明资金参与积极
- 抢筹/抢跑的态度坚决
- 信号可信度高

竞价量能小：
- 说明资金参与度低
- 观望情绪浓厚
- 信号可信度低
```

### 怎么做（操作）

**使用功能**：集合竞价、隔夜委托、消息面

**操作步骤**：

```
步骤1：隔夜单处理（8:30-8:45）
- 查看隔夜单变化
- 9:15-9:20观察撤单情况
- 9:20-9:25决定是否撤单

信号判断：
┌────────────────────────────────────────────────────────┐
│ 信号      │  含义      │  操作                      │
├────────────────────────────────────────────────────────┤
│ 抢筹     │  隔夜单大增 │  排队靠前                  │
│ 抢跑     │  隔夜单大减 │  及时撤单                  │
└────────────────────────────────────────────────────────┘

步骤2：消息解读（8:45-9:00）
- 查看隔夜重要消息（政策、行业、个股）
- 解读对持仓/计划股的影响
- 决定是否调整计划

步骤3：竞价观察（9:00-9:15）
- 首页 → 点击「集合竞价」
- 观察计划股的竞价情况
- 记录竞价涨幅、量能

步骤4：最终确认（9:15-9:25）
┌────────────────────────────────────────────────────────┐
│ 今日是否开仓？                                       │
├────────────────────────────────────────────────────────┤
│ □ 昨日涨停表现好 → 可开仓                          │
│ □ 昨日跌停多 → 不开仓                              │
│ □ 竞价抢筹多 → 积极开仓                            │

├────────────────────────────────────────────────────────┤
│ 计划股是否按预期开？                                │
├────────────────────────────────────────────────────────┤
│ □ 符合 → 按计划买                                  │
│ □ 超预期 → 临时加仓                                │
│ □ 不及预期 → 放弃/观察                             │
└────────────────────────────────────────────────────────┘
```

## 4.2 盘中阶段（9:30-14:30）

### 做什么（任务）

1. 执行买入计划
2. 监控持仓股走势
3. 执行卖出计划
4. 盘中应变

### 为什么做（逻辑）

> **核心原理**：盘中是计划的执行阶段
>
> - 计划是用来执行的，不是用来看看的
> - 盘中唯一要做的事：符合计划就做，不符合就不做

**为什么要「计划你的交易」？**

```
没有计划的盘中：
- 涨了想追
- 跌了想割
- 到处是机会
- 到处都是坑

有计划的盘中：
- 符合条件就买
- 不符合就等
- 只做计划内的交易
```

### 怎么做（操作）

**使用功能**：异动监控、涨跌停监控、板块联动

**操作步骤**：

```
步骤1：开盘博弈（9:30-10:00）

- 观察资金流向：资金是否在持续流入
- 执行买入：符合计划条件 → 买入

步骤3：午间复盘（11:30-13:00）

# 第六部分：核心理念总结

## 6.1 十二字真言

```
1. 资金在哪我在哪
   - 资金流入 → 跟进
   - 资金流出 → 回避

2. 情绪好我就做
   - 高潮期 → 重仓
   - 退潮期 → 空仓

3. 情绪差我就走
   - 盘面危险 → 卖出
   - 计划失败 → 止损

4. 错了就止损
   - -5%必走
   - 不幻想
   - 不死扛
```

## 6.2 决策流程图

```
                    ┌─ 情绪指标 → 周期判断
                    │
复盘 ──────────────┼─ 涨停板 → 找龙头
                    │
                    ├─ 炸板分析 → 避风险
                    │
                    ├─ 龙虎榜 → 跟主力
                    │
                    └─ 资金流向 → 判趋势
                              │
                              ↓
                         ┌─ 仓位计划
                         │
计划 ────────────────┼─ 标的计划
                         │
                         ├─ 买入条件
                         │
                         └─ 止损止盈
                                  │
                                  ↓
                             ┌─ 盘前
                             │
执行 ─────────────────┼─ 盘中
                             │
                             └─ 盘后
```
