## 五、快捷使用说明

### 5.1 模板使用流程

```
每日复盘流程：
1. 收盘后打开模板
2. 填写基础数据（10分钟）
3. 复盘涨停板（20分钟）
4. 复盘炸板（10分钟）
5. 复盘龙虎榜（10分钟）
6. 复盘资金流向（10分钟）
7. 总结操作（10分钟）
8. 制定明日计划（10分钟）

总计：约80分钟
```

### 5.2 注意事项

1. **当日事当日毕**：复盘不要拖到第二天
2. **数据要准确**：数字要反复核对
3. **反思要深刻**：找到真正的问题
4. **计划要具体**：条件要明确可执行

### 5.3 模板获取

建议将模板复制到自己的笔记软件（Notion/飞书/有道云），方便随时调用和统计。

## 六、完整每日操作框架

> 做什么 → 为什么做 → 怎么做 → 判断标准是什么 → 为什么是这个标准 → 怎么定的标准

### 6.1 每日时间线总览

| 时间段 | 任务 | 核心目标 |
|--------|------|----------|
| 15:00-16:00 | 收盘后复盘 | 了解今天盘面 |
| 16:00-17:00 | 制定计划 | 明确明天买什么 |
| 08:30-09:30 | 开盘前准备 | 确认计划可行 |
| 09:20-09:30 | 竞价观察 | 判断开盘是否符合预期 |
| 09:30-11:30 | 上午交易 | 执行计划，管理持仓 |
| 11:30-13:00 | 午间复盘 | 复盘上午，调整下午 |
| 13:00-14:30 | 下午交易 | 等待机会，处理持仓 |
| 14:30-15:00 | 尾盘处理 | 不开新仓，只处理持仓 |

### 6.2 收盘后复盘（15:00-16:00）

#### 6.2.1 基础数据记录

**做什么**：记录今日市场基础数据

**为什么做**：数据是判断情绪周期的基础

**怎么做**：统计涨停家数、炸板率、跌停家数、最高板、昨日涨停溢价、昨日连板表现

**判断标准**：

| 项目 | 数值 | 含义 |
|------|------|------|
| 涨停家数 | >50 | 情绪高潮 |
| 涨停家数 | 30-50 | 情绪回暖 |
| 涨停家数 | 15-30 | 情绪分化 |
| 涨停家数 | <15 | 情绪冰点 |
| 炸板率 | <15% | 健康 |
| 炸板率 | 15-30% | 正常 |
| 炸板率 | >40% | 风险 |
| 跌停家数 | <5 | 健康 |
| 跌停家数 | >20 | 恐慌 |

#### 6.2.2 情绪周期判断

**做什么**：判断当前处于哪个周期阶段

**为什么做**：周期阶段决定仓位和策略

**周期阶段与策略**：

| 阶段 | 涨停 | 炸板率 | 跌停 | 最高板 | 策略 |
|------|------|--------|------|--------|------|
| 启动 | 15-30 | <30% | <10 | 2-3板 | 轻仓试错 |
| 发酵 | 30-50 | <25% | <5 | 4-5板 | 半仓参与 |
| 高潮 | >50 | <20% | <3 | 6板+ | 重仓龙头 |
| 分歧 | 30-50 | >35% | >10 | 回落 | 减仓锁利 |
| 退潮 | <30 | >40% | >15 | 下降 | 空仓观望 |
| 冰点 | <15 | >50% | >20 | 1-2板 | 等待转折 |

#### 6.2.3 涨停板复盘

**做什么**：复盘今日涨停股，找到主线和龙头

**重点关注条件**：
- 板块内前3只涨停
- 板块有3只以上涨停（板块强度 >50%）
- 流通市值 <100亿
- 涨停原因清晰（政策、业绩、消息）
- 昨日涨停溢价 >2%

#### 6.2.4 炸板复盘

**做什么**：复盘今日炸板股，避免明日踩坑

**炸板原因分类**：
- 跟风炸板：板块不够强，后续不看
- 板块回调：板块可能退潮，谨慎
- 量能不足：主力不坚决，观望
- 大盘拖累：系统性风险，减仓

#### 6.2.5 龙虎榜复盘

**做什么**：分析机构和游资动向

**判断标准**：

| 资金类型 | 大单标准 | 关注度 |
|----------|----------|--------|
| 机构买入 | >1亿 | 重点 |
| 游资买入 | >5000万 | 关注 |
| 机构+游资合力 | 同时买入 | 重中之重 |

#### 6.2.6 资金流向复盘

**做什么**：分析资金去了哪里，找到钱在哪里

**判断标准**：

| 流入 | 金额标准 | 信号 |
|------|----------|------|
| 主力净流入 | >10亿 | 大主线 |
| 主力净流入 | 5-10亿 | 支线 |
| 5日连续流入 | - | 趋势形成 |

### 6.3 制定计划（16:00-17:00）

#### 6.3.1 仓位计划

**做什么**：根据情绪周期确定明日仓位

| 周期阶段 | 建议仓位 |
|----------|----------|
| 启动 | 3成 |
| 发酵 | 5成 |
| 高潮 | 7成 |
| 分歧 | 3成 |
| 退潮 | 0-1成 |
| 冰点 | 1-2成 |

#### 6.3.2 标的计划

**做什么**：筛选明日重点标的

**标的筛选条件**：
- 板块龙头：前3只涨停
- 板块强度：>50%
- 资金认可：主力净流入 >3亿
- 形态良好：趋势向上，MACD金叉
- 题材正宗：涨停原因清晰

#### 6.3.3 买入计划

**做什么**：制定每个标的的具体买入条件

**判断标准**：

| 条件 | 标准 | 说明 |
|------|------|------|
| 竞价高开 | 2-5% | 太低不受关注，太高容易炸 |
| 竞价量能 | >前5日均量30% | 资金积极参与 |
| 大盘环境 | 无急跌风险 | 系统性安全 |
| 板块配合 | 板块整体强势 | 板块支持 |

#### 6.3.4 卖出计划

**做什么**：制定持仓股的止盈止损计划

**止盈止损标准**：

| 情况 | 操作 |
|------|------|
| 亏损 -5% | 无条件止损 |
| 盈利 10% | 卖1/3 |
| 盈利 15% | 再卖1/3 |
| 剩余1/3 | 跌破5日线全走 |
| 收盘跌破5日线 | 次日开盘走 |

### 6.4 开盘前准备（08:30-09:30）

#### 6.4.1 隔夜消息

**做什么**：浏览隔夜重要消息

**怎么做**：查看政策消息、外围市场走势、大宗商品走势、个股公告

#### 6.4.2 开盘准备

**做什么**：确认昨日计划是否需要调整

#### 6.4.3 竞价观察

**做什么**：观察竞价情况

**判断标准**：

| 竞价表现 | 判断 | 操作 |
|---------|------|------|
| 高开2-5%，量能足 | 符合预期 | 按计划买入 |
| 高开 >7% | 太高 | 放弃或等回调 |
| 低开 | 不及预期 | 观望或取消 |
| 平开 | 观察 | 等开盘后走势 |

### 6.5 盘中交易（09:30-15:00）

#### 6.5.1 买入执行

**买入时机**：开盘后5分钟内确认走势，符合条件立即买入

**买入价格与数量**：
- 价格：不追涨超过开盘价3%
- 数量：单票 ≤30%

**买入条件确认清单**：
- [ ] 竞价高开 ≥2%
- [ ] 竞价量能放大
- [ ] 板块整体强势
- [ ] 大盘无急跌风险
- [ ] 不是板块最后一个涨停

#### 6.5.2 持仓管理

**持仓监控**：

| 走势 | 操作 |
|------|------|
| 涨停 | 持有不动 |
| 炸板3分钟不回封 | 必须走 |
| 急跌 -3% | 准备止损 |
| 量能异常缩量 | 可能见顶 |

**动态止盈**：
- 盈利 >5%：止盈位上移到成本价
- 盈利 >10%：止盈位上移到5%
- 盈利 >15%：止盈位上移到10%
- 出现滞涨信号：主动止盈一半

#### 6.5.3 盘中选股

**条件**：
- 涨幅榜前20
- 首次涨停或二板
- 板块3只以上涨停
- 流通市值 <100亿

**盘中买入纪律**：
- 只能买10:00前涨停
- 等第一次回踩再买
- 单票 ≤20%
- 14:30前买入

### 6.6 午间复盘（11:30-13:00）

#### 6.6.1 上午盘面总结

**判断标准**：
- 涨停 >30家、炸板率 <30%：偏暖
- 涨停 <15家、炸板率 >40%：退潮

#### 6.6.2 下午策略调整

**仓位调整**：
- 情绪好：下午可加仓
- 情绪一般：保持仓位
- 情绪差：减仓或空仓

**持仓调整**：
- 亏损股 -3%以内：午后反弹走
- 亏损股 -3%以上：准备止损
- 2:00前处理完问题持仓
