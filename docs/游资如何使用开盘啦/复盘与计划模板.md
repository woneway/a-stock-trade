## 五、快捷使用说明

### 5.1 模板使用流程

```
每日复盘流程：
1. 收盘后打开模板
2. 填写基础数据（10分钟）
3. 复盘涨停板（20分钟）
4. 复盘炸板（10分钟）
5. 复盘龙虎榜（10分钟）
6. 复盘资金流向（10分钟）
7. 总结操作（10分钟）
8. 制定明日计划（10分钟）

总计：约80分钟
```

### 5.2 注意事项

1. **当日事当日毕**：复盘不要拖到第二天
2. **数据要准确**：数字要反复核对
3. **反思要深刻**：找到真正的问题
4. **计划要具体**：条件要明确可执行

### 5.3 模板获取

建议将模板复制到自己的笔记软件（Notion/飞书/有道云），方便随时调用和统计。
