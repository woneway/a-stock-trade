# 游资判断方法大全

> 本文档补充开盘啦使用过程中的各种具体判断方法
> 涵盖：情绪判断、板块判断、个股判断、买卖时机判断等

---

## 一、情绪周期判断方法

### 1.1 基础指标判断法

**核心指标**：

| 指标 | 计算方式 | 判断标准 |
|------|----------|----------|
| **涨停家数** | 当日涨停股数量 | >50强，20-50中，<20弱 |
| **炸板率** | 炸板数/涨停数 | <15%低，15-30%中，>30%高 |
| **跌停家数** | 当日跌停股数量 | 0-3低风险，>10高风险 |
| **最高板** | 最高连板数 | 5板+强，3-4板中，<3板弱 |
| **昨日溢价** | 昨日涨停股今日平均涨幅 | >3%强，0-3%中，<0%弱 |

**组合判断**：

| 组合 | 信号 | 策略 |
|------|------|------|
| 涨停多+炸板低+跌停少 | 强势 | 重仓 |
| 涨停少+炸板高+跌停多 | 弱势 | 空仓 |
| 涨停中+炸板中+跌停中 | 中性 | 半仓 |

---

### 1.2 梯队完整性判断法

**涨停梯队**（来自开盘啦官方）：

```
一进二：首板 → 二板
二进三：二板 → 三板
三进四：三板 → 四板
四进五：四板 → 五板
五进六+：五板以上
```

**判断标准**：

| 梯队 | 完整度 | 信号 |
|------|--------|------|
| 一进二~五进六 | 全部完整 | 主升期，强烈做多 |
| 一进二~三进四 | 完整 | 发酵期，积极做多 |
| 仅一进二 | 有 | 启动期，试错 |
| 断层 | 无 | 退潮期，空仓 |

**实战用法**：

```
1. 打开涨停板 → 查看各梯队数量
2. 如果5进6有3只以上 → 主升期
3. 如果2进3有10只以上 → 发酵期
4. 如果只有1进2 → 启动期
5. 如果断层 → 退潮期
```

---

### 1.3 量能配合判断法

**量价关系**（来自开盘啦官方）：

| 量能状态 | 大盘表现 | 信号 |
|----------|----------|------|
| 放量上涨 | 量增价涨 | 健康的上涨 |
| 放量下跌 | 量增价跌 | 恐慌出逃 |
| 缩量上涨 | 量减价涨 | 虚涨小心 |
| 缩量横盘 | 量缩价平 | 变盘前兆 |

**量能预测**（开盘啦特色功能）：

> 开盘啦可盘中实时预测当天沪深两市总成交额

| 预测量能 | 实际量能 | 判断 |
|----------|----------|------|
| 1.5万亿 | 1.8万亿 | 超预期，放量 |
| 1.5万亿 | 1.2万亿 | 不及预期，缩量 |

---

### 1.4 周期阶段综合判断

**六阶段判断清单**：

| 阶段 | 涨停 | 最高板 | 炸板率 | 跌停 | 梯队 | 操作 |
|------|------|--------|--------|------|------|------|
| **启动** | 20-30 | 2-3 | 25-35% | 5-10 | 有断层 | 试错龙头 |
| **发酵** | 30-50 | 3-4 | 15-25% | 2-5 | 较完整 | 积极参与 |
| **高潮** | 50+ | 5+ | 10-15% | 0-2 | 完整 | 持股待涨 |
| **分歧** | 30-50 | 4-5 | 25-35% | 5-15 | 断层 | 谨慎追高 |
| **退潮** | 10-20 | 2-3 | 35-45% | 15-30 | 混乱 | 空仓休息 |
| **冰点** | <10 | 1-2 | >45% | >30 | 无 | 等待 |

## 二、板块判断方法

### 2.1 板块强度判断

**判断流程**：

```
1. 资金流向 → 板块 → 主力净流入排行
2. 查看净流入前5板块
3. 判断是否有持续性：
   - 连续3天净流入 → 持续性强
   - 仅今日净流入 → 持续性弱
4. 判断是否与大盘共振：
   - 大盘涨，板块涨 → 共振，可追
   - 大盘涨，板块跌 → 分化，观望
```

**板块强度指标**：

| 指标 | 强 | 弱 |
|------|-----|-----|
| 板块涨幅 | >5% | <2% |
| 涨停数量 | >5只 | <2只 |
| 主力净流入 | >10亿 | <2亿 |
| 持续天数 | 3天+ | 1天 |

---

### 2.2 板块轮动判断

**轮动规律**：

```
1. 启动轮动：新题材出现 → 资金追捧
2. 扩散轮动：龙头涨太高 → 资金找低位跟风
3. 切换轮动：老题材结束 → 新题材启动
4. 补涨轮动：主线涨不动 → 冷门股补涨
```

**判断方法**：

```
1. 打开区间统计 → 近5日
2. 对比各板块涨幅
3. 涨幅靠前的板块：主线/热点
4. 涨幅靠后的板块：冷门/待补涨
5. 观察资金切换：
   - 资金从涨幅大的板块流出 → 可能见顶
   - 资金流入涨幅小的板块 → 可能切换
```

---

### 2.3 主线板块识别

**主线特征**（来自开盘啦官方）：

1. **资金持续净流入**
   - 连续3天以上净流入
   - 净流入金额大（10亿+）

2. **板块效应强**
   - 涨停股数量多（5只+）
   - 联动性好（跟风股跟涨）

3. **与大盘共振**
   - 大盘涨，板块涨更多
   - 大盘跌，板块相对抗跌

**识别步骤**：

```
1. 资金流向 → 板块 → 主力净流入
2. 筛选连续3天净流入的板块
3. 对比板块内涨停股数量
4. 查看区间涨幅是否领先
5. 确认：主线板块 = 资金持续流入 + 涨停多 + 涨幅大
```

## 三、个股判断方法

### 3.1 龙头股识别

**龙头股特征**：

| 特征 | 说明 | 重要性 |
|------|------|--------|
| 最高板 | 连板数最高 | ★★★★★ |
| 率先涨停 | 板块内第一个涨停 | ★★★★★ |
| 封单最大 | 涨停封单金额最大 | ★★★★☆ |
| 带动性强 | 涨停带动板块 | ★★★★☆ |
| 抗跌 | 大盘跌时不跌 | ★★★☆☆ |

**判断步骤**：

```
1. 涨停板 → 筛选连板
2. 按连板数排序 → 最高的是龙头
3. 查看是否率先涨停
4. 查看封单金额
5. 观察次日反馈：继续涨停 = 真龙头
```

---

### 3.2 跟风股判断

**跟风股特征**：

| 特征 | 说明 | 风险 |
|------|------|------|
| 后于龙头涨停 | 龙头先涨停，跟风后涨停 | 中 |
| 封单小 | 封单金额小 | 高 |
| 助攻龙头 | 涨停为了稳定龙头 | 中 |
| 带动性弱 | 不带动板块 | 低 |

**判断标准**：

```
跟风股 = 板块内非龙头涨停股
- 涨停时间晚于龙头
- 封单金额小于龙头
- 涨幅通常小于龙头
```

**操作建议**：

| 类型 | 操作 | 风险 |
|------|------|------|
| 龙头 | 持有/低吸 | 低 |
| 跟风首板 | 可打 | 中 |
| 跟风二板 | 谨慎 | 高 |
| 跟风三板+ | 不买 | 极高 |

---

### 3.3 炸板股风险判断

**炸板原因分类**：

| 原因 | 特征 | 风险 | 能否买 |
|------|------|------|--------|
| 跟风炸板 | 龙头已封住，跟风被炸 | 高 | 否 |
| 板块炸板 | 板块整体回落 | 高 | 否 |
| 量能不足 | 封板资金不够 | 中 | 观察 |
| 大盘拖累 | 大盘大跌拖累 | 中 | 等修复 |
| 主力试盘 | 试盘后回落 | 低 | 可能 |

**判断步骤**：

```
1. 炸板分析 → 查看炸板股列表
2. 看炸板原因：
   - 跟风炸板 → 放弃
   - 板块炸板 → 规避板块
3. 看炸板时间：
   - 早盘炸板 → 弱
   - 尾盘炸板 → 强
4. 看回落幅度：
   - 回落>5% → 弱
   - 回落<2% → 强
```

---

### 3.4 反包股判断

**反包条件**：

| 条件 | 说明 |
|------|------|
| 昨日炸板 | 昨天曾经涨停 |
| 今天高开 | 今日高开超2% |
| 快速涨停 | 上午涨停 |
| 量能配合 | 成交量放大 |

**反包概率**：

| 情况 | 反包概率 |
|------|----------|
| 龙头反包 | 60%+ |
| 跟风反包 | 30% |
| 板块反包 | 40% |
| 大盘弱势 | <10% |

## 四、买卖时机判断方法

### 4.1 买入时机判断

**竞价买入判断**：

| 竞价形态 | 涨幅 | 信号 | 能否买 |
|----------|------|------|--------|
| 抢筹 | >5% | 强势 | 可买（风险高） |
| 强势 | 2-5% | 正常 | 等回调买 |
| 平开 | 0-2% | 观望 | 观察 |
| 弱势 | -2-0% | 弱 | 不买 |
| 抢跑 | <-2% | 极弱 | 放弃 |

**盘中买入判断**：

| 信号 | 说明 | 买入 |
|------|------|------|
| 龙头首次分歧 | 龙头第一次炸板/回落 | 低吸 |
| 板块首次爆发 | 板块第一次批量涨停 | 追最强 |
| 跟风启动 | 跟风股直线拉升 | 立即买 |
| 弱转强 | 低开高走突破分时 | 可以买 |

**分时买入信号**：

```
1. 突破分时均线 → 买入
2. 放量拉升 → 买入
3. 封板瞬间 → 排队买
4. 回调不破均线 → 低吸
```

---

### 4.2 卖出时机判断

**止损判断**：

| 情况 | 止损线 | 必须执行 |
|------|--------|----------|
| 单笔亏损 | -5% | 是 |
| 收盘跌破5日线 | 收盘价 | 是 |
| 大盘急跌 | -2% | 是 |
| 板块大跌 | -3% | 是 |

**止盈判断**：

| 情况 | 止盈线 | 操作 |
|------|--------|------|
| 盈利>10% | 10% | 考虑分批卖 |
| 出现滞涨 | 最高点-3% | 卖出 |
| 跌破5日线 | 收盘价 | 卖出 |
| 板块走弱 | 板块下跌 | 卖出 |

**卖点信号**：

```
1. 涨停被炸 → 卖出半仓
2. 放量不涨停 → 卖出
3. 破分时均线 → 卖出
4. 大盘急跌 → 卖出
5. 达到目标价 → 卖出
```

---

### 4.3 仓位判断方法

**根据周期定仓位**：

| 周期 | 仓位 | 说明 |
|------|------|------|
| 启动 | 5-7成 | 试错 |
| 发酵 | 7-9成 | 积极参与 |
| 高潮 | 8-10成 | 持股待涨 |
| 分歧 | 3-5成 | 谨慎 |
| 退潮 | 0-2成 | 空仓 |
| 冰点 | 0成 | 等待 |

**根据盘面定仓位**：

| 盘面 | 仓位调整 |
|------|----------|
| 涨停50+ | +2成 |
| 涨停20-50 | 不变 |
| 涨停<20 | -2成 |
| 跌停10+ | -3成 |

## 五、资金流向判断方法

### 5.1 主力资金判断

**资金性质**：

| 类型 | 特征 | 操作参考 |
|------|------|----------|
| 机构资金 | 买入稳定，持续性强 | 中线持有 |
| 游资资金 | 买入激进，短线爆发 | 快进快出 |
| 合力资金 | 机构+游资共振 | 重点关注 |

**判断步骤**：

```
1. 龙虎榜 → 机构买入
2. 对比机构vs游资买入
3. 机构为主 → 中线
4. 游资为主 → 短线
5. 合力 → 重点
```

---

### 5.2 资金持续性判断

**持续性判断**：

| 形态 | 说明 | 信号 |
|------|------|------|
| 持续净流入 | 5日+连续流入 | 主线 |
| 首次净流入 | 今日首次大幅流入 | 启动 |
| 脉冲净流入 | 偶尔大幅流入 | 不稳 |
| 持续净流出 | 5日+连续流出 | 回避 |

**实战用法**：

```
1. 资金流向 → 区间统计 → 近5日
2. 筛选持续净流入的板块
3. 筛选持续净流入的个股
4. 持续性 > 脉冲性
```

---

### 5.3 资金强度判断

**强度分级**：

| 强度 | 净流入 | 操作 |
|------|--------|------|
| 强力 | >10亿 | 重点关注 |
| 中力 | 5-10亿 | 关注 |
| 弱力 | 1-5亿 | 观察 |
| 无力 | <1亿 | 忽略 |

## 六、盘口判断方法

### 6.1 集合竞价盘口

**竞价量能**：

| 量能 | 信号 |
|------|------|
| 放大 | 资金活跃 |
| 缩量 | 资金观望 |

**竞价走势**：

| 走势 | 信号 |
|------|------|
| 持续走高 | 抢筹 |
| 持续走低 | 抢跑 |
| 震荡 | 观望 |

---

### 6.2 分时盘口

**量价配合**：

| 形态 | 信号 |
|------|------|
| 放量上涨 | 健康的涨 |
| 缩量上涨 | 虚涨 |
| 放量下跌 | 真跌 |
| 缩量下跌 | 观望 |

**大单观察**：

| 信号 | 含义 |
|------|------|
| 持续大单买入 | 主力吸筹 |
| 大单对倒 | 可能出货 |
| 大单卖出 | 主力跑路 |

---

### 6.3 涨停盘口

**封板质量**：

| 指标 | 好 | 差 |
|------|-----|-----|
| 封单金额 | >1亿 | <1000万 |
| 封单稳定性 | 稳定 | 反复打开 |
| 撤单情况 | 少 | 频繁 |
| 跟风盘 | 多 | 少 |

**开板预警**：

```
1. 封单持续减少 → 可能开板
2. 大单卖出增多 → 可能开板
3. 跟风股回落 → 可能开板
4. 大盘急跌 → 可能开板
```

## 七、综合决策清单

### 7.1 买入决策检查

```
□ 情绪周期在启动/发酵/高潮期？
□ 是主线板块？
□ 是板块龙头？
□ 竞价符合预期？
□ 大盘环境配合？
□ 有新题材/政策催化？
□ 符合买入模式？
□ 止损位设定？
```

全部满足 → 买入

---

### 7.2 卖出决策检查

```
□ 达到止损线？→ 必须卖
□ 达到止盈线？→ 考虑卖
□ 出现滞涨？→ 卖出
□ 板块走弱？→ 卖出
□ 大盘风险？→ 卖出
□ 有更好标的？→ 换股
```

出现任一情况 → 考虑卖出

---

### 7.3 空仓决策检查

```
□ 涨停<20家？→ 考虑空仓
□ 最高板<3板？→ 考虑空仓
□ 炸板率>30%？→ 考虑空仓
□ 跌停>10家？→ 建议空仓
□ 周期在退潮/冰点？→ 建议空仓
```

出现任一情况 → 建议空仓

## 八、常见判断误区

### 8.1 情绪判断误区

| 误区 | 纠正 |
|------|------|
| 涨停多就能买 | 看涨停的是什么股 |
| 跌停少就安全 | 看是什么在跌 |
| 最高板高就强 | 看是否开始A杀 |

### 8.2 板块判断误区

| 误区 | 纠正 |
|------|------|
| 板块涨就能买 | 要买龙头，跟风危险 |
| 资金流入就能买 | 持续流入才可靠 |
| 板块大就能买 | 要看资金认可度 |

### 8.3 个股判断误区

| 误区 | 纠正 |
|------|------|
| 涨停快就能买 | 要看是否是龙头 |
| 低价就能买 | 要看题材和资金 |
| 图形好就能买 | 要看市场周期 |

## 九、总结：判断方法体系

```
                    ┌─ 基础指标判断
                    │
                    ├─ 梯队完整性判断
情绪判断 ─────────┼─ 周期阶段判断
                    │           ┌─ 资金持续性
                    └─ 量能判断 ┤
                                 ├─ 资金强度
                                 └─ 资金性质

板块判断 ─────────┼─ 板块强度判断
                    ├─ 轮动判断
                    └─ 主线识别

                    ┌─ 龙头识别
个股判断 ─────────┼─ 跟风识别
                    ├─ 风险识别
                    └─ 反包判断

买卖时机 ─────────┼─ 买入时机
                    ├─ 卖出时机
                    └─ 仓位判断
```

**核心原则**：
1. 先判断情绪，再决定仓位
2. 先看板块，再选个股
3. 先选龙头，再买跟风
4. 先设止损，再谈盈利
